ipayload_formatter module¶

class ipayload_formatter¶

Bases: object

ipayload_formatter is the interface class for Json. This includes methods for converting Json to nitro resource and vice versa. Provides classes necessary to create base_resource, base_response, Json, login, logout and reboot objects.
