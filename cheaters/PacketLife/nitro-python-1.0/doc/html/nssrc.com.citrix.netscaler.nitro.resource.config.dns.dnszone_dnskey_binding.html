class dnszone_dnskey_binding¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Binding class showing the dnskey that can be bound to dnszone.

zonename¶

Name of the zone. Mutually exclusive with the type parameter.<br/>Minimum length = 1.

keyname¶

Name of the public/private DNS key pair with which to sign the zone. You can sign a zone with up to four keys.<br/>Minimum length = 1.

siginceptiontime¶

The time when sign was done with this key.<br/>Minimum value = 1.

signed¶

Integer which denote status of keys.

expires¶

Time period for which to consider the key valid, after the key is used to sign a zone.
