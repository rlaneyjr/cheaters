server_service_binding module¶

class server_service_binding¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Binding class showing the service that can be bound to server.

name¶

Name of the server for which to display parameters.<br/>Minimum length = 1.

servicename¶

The services attatched to the server.

serviceipstr¶

This field has been intorduced to show the dbs services ip.

svctype¶

The type of bound service.<br/>Possible values = HTTP, FTP, TCP, UDP, SSL, SSL_BRIDGE, SSL_TCP, DTLS, NNTP, RPCSVR, DNS, ADNS, SNMP, RTSP, DHCPRA, ANY, SIP_UDP, DNS_TCP, ADNS_TCP, MYSQL, MSSQL, ORACLE, RADIUS, RDP, DIAMETER, SSL_DIAMETER, TFTP.

svrstate¶

The state of the bound service.<br/>Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED.

port¶

The port number to be used for the bound service.<br/>Range 1 - 65535.

serviceipaddress¶

The IP address of the bound service.
