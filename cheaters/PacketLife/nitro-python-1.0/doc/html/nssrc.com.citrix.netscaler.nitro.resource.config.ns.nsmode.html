nsmode module¶

class nsmode¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for ns mode resource.

mode¶

Mode to be enabled. Multiple modes can be specified by providing a blank space between each mode.<br/>Possible values = FR, FastRamp, L2, L2mode, L3, L3mode, USIP, UseSourceIP, CKA, ClientKeepAlive, TCPB, TCPBuffering, MBF, MACbasedforwarding, Edge, USNIP, SRADV, DRADV, IRADV, SRADV6, DRADV6, PMTUD, RISE_APBR, RISE_RHI, BridgeBPDUs.

fr¶

Fast Ramp.

l2¶

Layer 2 mode.

usip¶

Use Source IP.

cka¶

Client Keep-alive.

tcpb¶

TCP Buffering.

mbf¶

MAC-based forwarding.

edge¶

Edge configuration.
