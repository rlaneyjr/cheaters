class authenticationradiusaction¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for RADIUS action resource.

name¶

Name for the RADIUS action. Must begin with a letter, number, or the underscore character (_), and must contain only letters, numbers, and the hyphen (-), period (.) pound (#), space ( ), at (@), equals (=), colon (:), and underscore characters. Cannot be changed after the RADIUS action is added.<br/>Minimum length = 1.

serverip¶

IP address assigned to the RADIUS server.<br/>Minimum length = 1.

servername¶

RADIUS server name as a FQDN. Mutually exclusive with RADIUS IP address.<br/>Minimum length = 1.

serverport¶

Port number on which the RADIUS server listens for connections.<br/>Minimum length = 1.

authtimeout¶

Number of seconds the NetScaler appliance waits for a response from the RADIUS server.<br/>Default value: 3<br/>Minimum length = 1.

radkey¶

Key shared between the RADIUS server and the NetScaler appliance. Required to allow the NetScaler appliance to communicate with the RADIUS server.<br/>Minimum length = 1.

radnasip¶

If enabled, the NetScaler appliance IP address (NSIP) is sent to the RADIUS server as the Network Access Server IP (NASIP) address. The RADIUS protocol defines the meaning and use of the NASIP address.<br/>Possible values = ENABLED, DISABLED.

radnasid¶

If configured, this string is sent to the RADIUS server as the Network Access Server ID (NASID).

radvendorid¶

RADIUS vendor ID attribute, used for RADIUS group extraction.<br/>Minimum length = 1.

radattributetype¶

RADIUS attribute type, used for RADIUS group extraction.<br/>Minimum length = 1.

radgroupsprefix¶

RADIUS groups prefix string. This groups prefix precedes the group names within a RADIUS attribute for RADIUS group extraction.

radgroupseparator¶

RADIUS group separator string The group separator delimits group names within a RADIUS attribute for RADIUS group extraction.

passencoding¶

Encoding type for passwords in RADIUS packets that the NetScaler appliance sends to the RADIUS server.<br/>Default value: pap<br/>Possible values = pap, chap, mschapv1, mschapv2.

ipvendorid¶

Vendor ID of the intranet IP attribute in the RADIUS response. NOTE: A value of 0 indicates that the attribute is not vendor encoded.

ipattributetype¶

Remote IP address attribute type in a RADIUS response.<br/>Minimum length = 1.

accounting¶

Whether the RADIUS server is currently accepting accounting messages.<br/>Possible values = ON, OFF.

pwdvendorid¶

Vendor ID of the attribute, in the RADIUS response, used to extract the user password.<br/>Minimum length = 1.

pwdattributetype¶

Vendor-specific password attribute type in a RADIUS response.<br/>Minimum length = 1.

defaultauthenticationgroup¶

This is the default group that is chosen when the authentication succeeds in addition to extracted groups.<br/>Maximum length = 64.

callingstationid¶

Send Calling-Station-ID of the client to the RADIUS server. IP Address of the client is sent as its Calling-Station-ID.<br/>Default value: DISABLED<br/>Possible values = ENABLED, DISABLED.

ipaddress¶

IP address.
