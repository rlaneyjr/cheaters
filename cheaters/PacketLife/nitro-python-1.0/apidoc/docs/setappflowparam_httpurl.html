setappflowparam_httpurl

Use this method to set include the http URL that the NetScaler appliance received from the client.

Syntax



Parameters

httpurl

Include the http URL that the NetScaler appliance received from the client.
Default value = DISABLED.
Possible Values : ENABLED, DISABLED.

Return Value

Returns simpleResult
