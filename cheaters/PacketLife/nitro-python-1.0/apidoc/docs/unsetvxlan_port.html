unsetvxlan_port

Specifies UDP destination port for VXLAN packets.

Syntax



Parameters

id

Name of the networkvxlan.
Default value = 0.
Minimum value = 1.
Maximum value = 16777215.

port

Indicates that port will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
