Integrated Caching

integrated caching statistics. the integrated caching feature is used to cache static and dynamic web application data.

Statistics APIs

statcache

Use this method to shows Integrated Cache performance statistics.

statcachepolicy

Use this method to displays a summary of cache policy statistics.

statcachepolicylabel

Use this method to displays statistics of cache policy label(s).

statcachecontentgroup

Use this method to displays a summary of cache group statistics.

statcachepolicybinding

Use this method to display statistics of cache policy binding(s).
