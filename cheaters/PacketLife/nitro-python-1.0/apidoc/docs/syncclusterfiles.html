syncclusterfiles

Use this method to synchronizes SSL Certificates, SSL CRL lists, SSL VPN bookmarks, and other files from the configuration coordinator (CCO) to the other cluster nodes. Execute this method from the cluster IP address only. This method is automatically triggered from the CCO when a new node is added to a cluster and periodically triggered to synchronize updated files between the cluster nodes.
Note: Files on non-CCO nodes are not deleted if they do no not exist on the CCO.
