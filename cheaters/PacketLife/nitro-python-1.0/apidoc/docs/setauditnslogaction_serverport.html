setauditnslogaction_serverport

Use this method to set port on which the nslog server accepts connections.

Syntax



Parameters

name

Name of the nslog action to be modified.
This is mandatory parameter.

serverport

Port on which the nslog server accepts connections.
Minimum value = 1.

Return Value

Returns simpleResult
