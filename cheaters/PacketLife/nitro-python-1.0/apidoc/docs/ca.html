Ca

content accelerator configuration. the content accelerator feature is used to cache static and dynamic web application data.

Configuration APIs

addcaaction

Use this method to creates a content adapation action. This action must later be invoked in the 'add ca policy' method.

addcapolicy

Use this method to creates a content adaptation policy. This policy must later be invoked globally or at a content switching or load balancing virtual server.

bindcaglobal_policy

Use this method to bind policy to ca global.

getcaaction

Use this method to get information about a content adaptation action. If no name is specified, this method displays information of all available content adaptation actions.

getcaglobal

Use this method to get the content adaptation policies that are globally-bound to the NetScaler appliance.

getcapolicy

Use this method to get information about a content adaptation policy. If no name is specified, this method displays information of all available content adaptation policies.

renamecaaction

Use this method to renames a Content Accelerator action.

renamecapolicy

Use this method to renames content accelerator policy.

rmcaaction

Use this method to removes a ca action.

rmcapolicy

Use this method to removes a content adaptation policy.

setcaaction_accumressize

Use this method to set size of the data, in KB, that the server must respond with. The NetScaler uses this data to compute a hash which is then used to lookup within the T2100 appliance.

setcaaction_comment

Use this method to set information about the content adaptation action.

setcaaction_lbvserver

Use this method to set name of the load balancing virtual server that has the T2100 appliances as services.

setcaaction_type

Use this method to set specifies whether the NetScaler must lookup for the response on the T2100 appliance or serve the response directly from the server.

setcapolicy_action

Use this method to set name of content adaptation action to be executed when the rule is evaluated to true.

setcapolicy_comment

Use this method to set information about the content adaptation policy.

setcapolicy_logaction

Use this method to set name of messagelog action to use when a request matches this policy.

setcapolicy_rule

Use this method to set expression that determines which requests or responses match the content adaptation policy. When specifying the rule in the CLI, the description must be enclosed within double quotes.

setcapolicy_undefaction


unbindcaglobal_policy

Use this method to unbind policy from ca global.

unbindcaglobal_policynameEx

Use this method to unbind policy from ca global.

unsetcaaction_accumressize

Remove ca action accumressize setting.

unsetcaaction_comment

Remove ca action comment setting.

unsetcaaction_lbvserver

Remove ca action lbvserver setting.

unsetcaaction_type

Remove ca action type setting.

unsetcapolicy_comment

Remove ca policy comment setting.

unsetcapolicy_logaction

Remove ca policy logaction setting.

unsetcapolicy_undefaction

Remove ca policy undefaction setting.
