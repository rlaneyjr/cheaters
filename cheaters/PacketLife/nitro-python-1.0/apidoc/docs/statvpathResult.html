statvpathResult Structure Definition

The statvpathResult structure defines the return type for statvpath API.

Syntax



Members

rc

If the method succeeds, rc is 0 else rc > 0. Values above 0x8000 indicate Warnings.

message

If the method succeeds, message is NULL else message contains Error/Warning message.

vpathList

Pointer to the vpath List.
