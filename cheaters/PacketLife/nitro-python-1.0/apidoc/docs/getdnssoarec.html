getdnssoarec

Use this method to get the parameters of the specified Start of Authority (SOA) record. If no domain name is specified, all SOA records are displayed.

Syntax



Parameters

domain

The domain name.

type

Type of records to display. Available settings function as follows: * ADNS - Display all authoritative address records. * PROXY - Display all proxy address records. * ALL - Display all address records.

Return Value

Returns getdnssoarecResult
