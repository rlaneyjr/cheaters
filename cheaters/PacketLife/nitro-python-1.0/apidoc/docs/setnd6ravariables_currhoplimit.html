setnd6ravariables_currhoplimit

Use this method to set current Hop limit.

Syntax



Parameters

vlan

The VLAN number.
This is mandatory parameter.
Minimum value = 0.
Maximum value = 4094.

currhoplimit

Current Hop limit.
Default value = 64.
Minimum value = 0.
Maximum value = 255.

Return Value

Returns simpleResult
