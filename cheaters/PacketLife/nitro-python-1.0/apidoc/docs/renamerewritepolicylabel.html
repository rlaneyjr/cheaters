renamerewritepolicylabel

Use this method to renames a rewrite policy label.

Syntax



Parameters

labelname

Current name of the policy label.
This is mandatory parameter.

newname

New name for the rewrite policy label. Must begin with a letter, number, or the underscore character (_), and must contain only letters, numbers, and the hyphen (-), period (.) hash (#), space ( ), at (@), equals (=), colon (:), and underscore characters. The following requirement applies only to the NetScaler CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, "my policy label" or ‘my policy label’).
This is mandatory parameter.

Return Value

Returns simpleResult
