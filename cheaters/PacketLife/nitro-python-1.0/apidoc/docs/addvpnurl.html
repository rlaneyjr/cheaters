addvpnurl

Use this method to creates a bookmark link to an external or internal resource that appears on the Access Interface, according to type, as a web site link or file share link.

Syntax



Parameters

urlname

Name of the bookmark link.
This is mandatory parameter.

linkname

Description of the bookmark link. The description appears in the Access Interface.
This is mandatory parameter.

actualurl

Web address for the bookmark link.
This is mandatory parameter.

Return Value

Returns simpleResult
