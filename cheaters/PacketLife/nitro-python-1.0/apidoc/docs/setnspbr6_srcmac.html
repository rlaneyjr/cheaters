setnspbr6_srcmac

Use this method to set mAC address to match against the source MAC address of an outgoing IPv6 packet.

Syntax



Parameters

name

Name of the PBR6 whose parameters you want to modify.
This is mandatory parameter.

srcmac

MAC address to match against the source MAC address of an outgoing IPv6 packet.

Return Value

Returns simpleResult
