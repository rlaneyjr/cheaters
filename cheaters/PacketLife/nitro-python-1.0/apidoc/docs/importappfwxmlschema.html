importappfwxmlschema

Use this method to imports the specified XML Schema to the NetScaler appliance and assigns it the specified name.

Syntax



Parameters

src

URL (protocol, host, path, and file name) for the location at which to store the imported XML Schema. NOTE: The import fails if the object to be imported is on an HTTPS server that requires client certificate authentication for access.
This is mandatory parameter.

name

Name to assign to the XML Schema object on the NetScaler appliance.
This is mandatory parameter.

comment

Any comments to preserve information about the XML Schema object.

overwrite

Overwrite any existing XML Schema object of the same name.

Return Value

Returns simpleResult
