setauthenticationauthnprofile_authenticationlevel

Use this method to set authentication weight or level of the vserver to which this will bound. This is used to order TM vservers based on the protection required. A session that is created by authenticating against TM vserver at given level cannot be used to access TM vserver at a higher level.

Syntax



Parameters

name

Name of the authentication profile.
This is mandatory parameter.

authenticationlevel

Authentication weight or level of the vserver to which this will bound. This is used to order TM vservers based on the protection required. A session that is created by authenticating against TM vserver at given level cannot be used to access TM vserver at a higher level.
Maximum value = 255.

Return Value

Returns simpleResult
