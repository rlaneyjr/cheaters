setfeoparameter_jpegqualitypercent

Use this method to set the percentage value of a JPEG image quality to be reduced. Range: 0 - 100

Syntax



Parameters

jpegqualitypercent

The percentage value of a JPEG image quality to be reduced. Range: 0 - 100
Default value = 75.
Maximum value = 100.

Return Value

Returns simpleResult
