setautoscaleaction_quiettime

Use this method to set time in seconds no other policy is evaluated or action is taken

Syntax



Parameters

name

ActionScale action name.
This is mandatory parameter.

quiettime

Time in seconds no other policy is evaluated or action is taken
Default value = 300.

Return Value

Returns simpleResult
