Authorization

authorization configuration. authorization services check which resources users are authorized to access, and grant permissions accordingly.

Configuration APIs

addauthorizationpolicy

Use this method to creates an authorization policy.
Authorization policies allow AAA users and AAA groups to access resources through SSL VPN/AAA-TM enabled virtual servers.

addauthorizationpolicylabel

Use this method to creates a user-defined authorization policy label.

bindauthorizationpolicylabel_policy

Use this method to bind policy to authorization policylabel.

getauthorizationaction

Use this method to get details of authorization actions.

getauthorizationpolicy

Use this method to get the current settings for the specified authorization policy. If no policy name is provided, displays a list of all authorization policies currently configured on the NetScaler appliance.

getauthorizationpolicylabel

Use this method to get the current settings for the specified authorization policy label.

renameauthorizationpolicy

Use this method to rename a author policy.

renameauthorizationpolicylabel

Use this method to rename a auth policy label.

rmauthorizationpolicy

Use this method to removes an authorization policy.

rmauthorizationpolicylabel

Use this method to removes an authorization policy label.

setauthorizationpolicy_action

Use this method to set action to perform if the policy matches: either allow or deny the request.

setauthorizationpolicy_rule

Use this method to set name of the NetScaler named rule, or a default syntax expression, that the policy uses to perform the authentication.

unbindauthorizationpolicylabel_policy

Use this method to unbind policy from authorization policylabel.

unbindauthorizationpolicylabel_priority

Use this method to unbind priority from authorization policylabel.
