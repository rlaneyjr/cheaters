getappfwlearningdata

Use this method to get the unreviewed application firewall learning data for the specified profile and security check.

Syntax



Parameters

profilename

Name of the profile.
This is mandatory parameter.

securitycheck

Name of the security check.
This is mandatory parameter.

Return Value

Returns getappfwlearningdataResult
