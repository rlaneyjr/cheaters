setvpnclientlessaccessprofile_clientconsumedcookies

Use this method to set specify the name of the pattern set containing the names of the cookies, which are allowed between the client and the server. If a pattern set is not specified, NetSCaler Gateway does not allow any cookies between the client and the server. A cookie that is not specified in the pattern set is handled by NetScaler Gateway on behalf of the client.

Syntax



Parameters

profilename

Name of the clientless access profile to modify.
This is mandatory parameter.

clientconsumedcookies

Specify the name of the pattern set containing the names of the cookies, which are allowed between the client and the server. If a pattern set is not specified, NetSCaler Gateway does not allow any cookies between the client and the server. A cookie that is not specified in the pattern set is handled by NetScaler Gateway on behalf of the client.

Return Value

Returns simpleResult
