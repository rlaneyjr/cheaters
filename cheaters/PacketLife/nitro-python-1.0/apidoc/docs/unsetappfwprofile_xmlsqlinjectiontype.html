unsetappfwprofile_xmlsqlinjectiontype

Remove appfw profile xmlsqlinjectiontype setting.

Syntax



Parameters

name

Name of the appfwprofile.
Default value = 0.

xmlsqlinjectiontype

Indicates that xmlsqlinjectiontype will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
