unsetrnat_acl

Name of any configured extended ACL whose action is ALLOW. The condition specified in the extended ACL rule isused as the condition for the RNAT6 rule.

Syntax



Parameters

aclname

Indicates that aclname will be reverted or reset to the default value.

redirectport

Indicates that redirectport will be reverted or reset to the default value.
