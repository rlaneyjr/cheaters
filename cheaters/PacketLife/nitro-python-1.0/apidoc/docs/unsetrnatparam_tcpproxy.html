unsetrnatparam_tcpproxy

Enable TCP proxy, which enables the NetScaler appliance to optimize the RNAT TCP traffic by using Layer 4 features.

Syntax



Parameters

Indicates that tcpproxy will be reverted or reset to the default value.
Default value = 0.
