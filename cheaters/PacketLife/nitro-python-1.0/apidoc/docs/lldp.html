LLDP

link layer discovery protocol.

Configuration APIs

clearlldpneighbors

Use this method to removes LLDP neighbor info of interfaces

getlldpneighbors

Use this method to get Neighbor information per interface

getlldpparam

Use this method to get the global LLDP params

setlldpparam_holdtimetxmult

Use this method to set a multiplier for calculating the duration for which the receiving device stores the LLDP information in its database before discarding or removing it. The duration is calculated as the holdtimeTxMult (Holdtime Multiplier) parameter value multiplied by the timer (Timer) parameter value.

setlldpparam_mode

Use this method to set global mode of Link Layer Discovery Protocol (LLDP) on the NetScaler ADC. The resultant LLDP mode of an interface depends on the LLDP mode configured at the global and the interface levels.

setlldpparam_timer

Use this method to set interval, in seconds, between LLDP packet data units (LLDPDUs). that the NetScaler ADC sends to a directly connected device.

unsetlldpparam_holdtimetxmult

Remove lldp param holdtimetxmult setting.

unsetlldpparam_mode

Remove lldp param mode setting.

unsetlldpparam_timer

Remove lldp param timer setting.
