setdnsaction64_mappedrule

Use this method to set the expression to select the criteria for ipv4 addresses to be used for synthesis.
Only if the mappedrule is evaluated to true the corresponding ipv4 address is used for synthesis using respective prefix,
otherwise the A RR is discarded

Syntax



Parameters

actionname

Name of the dns64 action.
This is mandatory parameter.

mappedrule

The expression to select the criteria for ipv4 addresses to be used for synthesis. Only if the mappedrule is evaluated to true the corresponding ipv4 address is used for synthesis using respective prefix, otherwise the A RR is discarded

Return Value

Returns simpleResult
