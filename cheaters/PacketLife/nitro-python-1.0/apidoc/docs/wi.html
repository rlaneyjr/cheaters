WebInterface

web interface configuration.

Configuration APIs

addwisite

Use this method to creates a Web Interface site on the NetScaler appliance.
The NetScaler Web Interface feature provides access to Citrix XenApp and Citrix XenDesktop applications. Users access resources through a standard web browser or by using the Citrix XenApp plug-in.

bindwisite_accessmethod

Use this method to bind accessmethod to wi site.

bindwisite_farm

Use this method to bind farm to wi site.

bindwisite_farmnameEx

Use this method to bind farm to wi site.

bindwisite_translationinternalip

Use this method to bind translationinternalip to wi site.

getwisite

Use this method to get settings of all the Web Interface sites, or of a specified site. To display settings of all the Web Interface sites, run the method without any parameters.

installwipackage

Use this method to installs Web Interface and JRE tar files on the NetScaler appliance.

rmwisite

Use this method to removes a Web Interface site from the NetScaler appliance.

setwisite_agauthenticationmethod

Use this method to set method for authenticating a Web Interface site if you have specified Web Interface as the authentication point.
Available settings function as follows:
* Explicit - Users must provide a user name and password to log on to the Web Interface.
* Anonymous - Users can log on to the Web Interface without providing a user name and password. They have access to resources published for anonymous users.

setwisite_agurl

Use this method to set uRL of the Access Gateway.

setwisite_appwelcomemessage

Use this method to set specifies localized text to appear at the top of the main content area of the Applications screen. LanguageCode is en, de, es, fr, ja, or any other supported language identifier.

setwisite_authenticationpoint

Use this method to set authentication point for the Web Interface site.

setwisite_defaultaccessmethod

Use this method to set default access method for clients accessing the Web Interface site.

Note: Before you configure an access method based on the client IP address, you must enable USIP mode on the Web Interface service to make the client’s IP address available with the Web Interface.
Depending on whether the Web Interface site is configured to use an HTTP or HTTPS virtual server or to use access gateway, you can send clients or access gateway the IP address, or the alternate address, of a XenApp or XenDesktop server. Or, you can send the IP address translated from a mapping entry, which defines mapping of an internal address and port to an external address and port.
Note: In the NetScaler method line, mapping entries can be created by using the bind wi site method.

setwisite_defaultcustomtextlocale

Use this method to set default language for the Web Interface site.

setwisite_domainselection

Use this method to set domain names listed on the login screen for explicit authentication.

setwisite_footertext

Use this method to set localized text that appears in the footer area of all pages.

setwisite_hidedomainfield

Use this method to set the HideDomainField setting is used to control whether the domain field is displayed on the logon screen.

setwisite_kioskmode

Use this method to set user settings do not persist from one session to another.

setwisite_logindomains

Use this method to set [List of NetBIOS domain names], Domain names to use for access restriction.
Only takes effect when used in conjunction with the RestrictDomains setting.

setwisite_loginsysmessage

Use this method to set localized text that appears at the bottom of the main content area of the login screen.

setwisite_logintitle

Use this method to set a custom login page title for the Web Interface site.

setwisite_preloginbutton

Use this method to set localized text that appears as the name of the pre-login message confirmation button.

setwisite_preloginmessage

Use this method to set localized text that appears on the pre-login message page.

setwisite_prelogintitle

Use this method to set localized text that appears as the title of the pre-login message page.

setwisite_publishedresourcetype

Use this method to set method for accessing the published XenApp and XenDesktop resources.
Available settings function as follows:
* Online - Allows applications to be launched on the XenApp and XenDesktop servers.
* Offline - Allows streaming of applications to the client.
* DualMode - Allows both online and offline modes.

setwisite_restrictdomains

Use this method to set the RestrictDomains setting is used to enable/disable domain restrictions. If domain restriction is enabled, the LoginDomains list is used for validating the login domain. It is applied to all the authentication methods except Anonymous for XenApp Web and XenApp Services sites

setwisite_secondstaurl

Use this method to set uRL of the second Secure Ticket Authority (STA) server.

setwisite_sessionreliability

Use this method to set enable session reliability through Access Gateway.

setwisite_showrefresh

Use this method to set provides the Refresh button on the applications screen.

setwisite_showsearch

Use this method to set enables search option on XenApp websites

setwisite_staurl

Use this method to set uRL of the Secure Ticket Authority (STA) server.

setwisite_userinterfacebranding

Use this method to set specifies whether the site is focused towards users accessing applications or desktops. Setting the parameter to Desktops changes the functionality of the site to improve the experience for XenDesktop users. Citrix recommends using this setting for any deployment that includes XenDesktop.

setwisite_userinterfacelayouts

Use this method to set specifies whether or not to use the compact user interface.

setwisite_usetwotickets

Use this method to set request tickets issued by two separate Secure Ticket Authorities (STA) when a resource is accessed.

setwisite_websessiontimeout

Use this method to set time-out, in minutes, for idle Web Interface browser sessions. If a client's session is idle for a time that exceeds the time-out value, the NetScaler appliance terminates the connection.

setwisite_welcomemessage

Use this method to set localized welcome message that appears on the welcome area of the login screen.

setwisite_wiauthenticationmethods

Use this method to set the method of authentication to be used at Web Interface

setwisite_wiuserinterfacemodes

Use this method to set appearance of the login screen.
* Simple - Only the login fields for the selected authentication method are displayed.
* Advanced - Displays the navigation bar, which provides access to the pre-login messages and preferences screens.

unbindwisite_clientipaddress

Use this method to unbind clientipaddress from wi site.

unbindwisite_farm

Use this method to unbind farm from wi site.

unbindwisite_translationinternalip

Use this method to unbind translationinternalip from wi site.

uninstallwipackage

Use this method to removes the Web Interface and JRE tar files, and the entire Web Interface related configuration, from the NetScaler appliance.
