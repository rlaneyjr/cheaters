Network

networking commands.

Statistics APIs

statvlan

Use this method to display statistics for VLAN(s).

statinat

Use this method to display statistics for inat sessions.

statinatsession

Use this method to display statistics for stateful inat sessions.

statinterface

Use this method to displays the statistics of all interfaces or of the specified interface on the NetScaler appliance. To display the statistics of all interfaces, run the method without any parameters. To display the statistics of a particular interface, specify the ID of the interface.

statrnat

Use this method to display statistics for rnat sessions.

statbridge

Use this method to display bridging statistics.

statrnatip

Use this method to display statistics for RNAT sessions.

statnat64

Use this method to display statistics for nat64 sessions.

stattunnelip

Use this method to display the statistics related to IP tunnel.

stattunnelip6

Use this method to display the statistics related to IP tunnel.

statvpath

Use this method to display vPath statistics.

statvxlan

Use this method to display statistics for VXLAN(s).
