setinat_privateip

Use this method to set iP address of the server to which the packet is sent by the NetScaler. Can be an IPv4 or IPv6 address.

Syntax



Parameters

name

The name of the Inbound NAT (INAT) entry that you want to modify.
This is mandatory parameter.

privateip

IP address of the server to which the packet is sent by the NetScaler. Can be an IPv4 or IPv6 address.
Minimum value = 1.

Return Value

Returns simpleResult
