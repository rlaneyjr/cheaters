setwisite_showrefresh

Use this method to set provides the Refresh button on the applications screen.

Syntax



Parameters

sitepath

Path to the Web Interface site being created on the NetScaler appliance.
This is mandatory parameter.

showrefresh

Provides the Refresh button on the applications screen.
Default value = OFF.
Possible Values : ON, OFF.

Return Value

Returns simpleResult
