setnstcpprofile_kaprobeinterval

Use this method to set time interval, in seconds, before the next keep-alive (KA) probe, if the peer does not respond.

Syntax



Parameters

name

Name of the TCP profile to be modified.
This is mandatory parameter.

kaprobeinterval

Time interval, in seconds, before the next keep-alive (KA) probe, if the peer does not respond.
Default value = NSTCP_KA_DEFAULT_INTERVAL.
Minimum value = 1.
Maximum value = 4095.

Return Value

Returns simpleResult
