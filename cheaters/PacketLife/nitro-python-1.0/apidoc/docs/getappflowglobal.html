getappflowglobal

Use this method to get the AppFlow global bind points and the number of policies bound to each global bind point, or more detailed information about the specified bind point.

Syntax



Parameters

type

Global bind point for which to show detailed information about the policies bound to the bind point.

Return Value

Returns getappflowglobalResult
