unsigndnszone

Use this method to unsigns the specified DNS zone with the specified DNS key.

Syntax



Parameters

zonename

Name of the zone.
This is mandatory parameter.

keyname

Name of the public-private DNS key pair with which to unsign the zone.

Return Value

Returns simpleResult
