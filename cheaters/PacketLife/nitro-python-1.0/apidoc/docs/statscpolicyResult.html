statscpolicyResult Structure Definition

The statscpolicyResult structure defines the return type for statscpolicy API.

Syntax



Members

rc

If the method succeeds, rc is 0 else rc > 0. Values above 0x8000 indicate Warnings.

message

If the method succeeds, message is NULL else message contains Error/Warning message.

scpolicyList

Pointer to the scpolicy List.
