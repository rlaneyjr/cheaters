Appqoe

application level quality of experience configuration

Configuration APIs

addappqoeaction

Use this method to add a new AppQoE action for triggering

addappqoepolicy

Use this method to add a new AppQoE policy for binding rule with action

getappqoeaction

Use this method to get configured AppQoE action(s).

getappqoecustomresp

Use this method to get lists all HTML page objects on the NetScaler appliance.

getappqoeparameter

Use this method to get the values of the session life and filename parameters

getappqoepolicy

Use this method to get all the configured AppQoE policies.

importappqoecustomresp

Use this method to downloads the input HTML Page to NetScaler Box with the given object name

rmappqoeaction

Use this method to removes the specified AppQoE action.

rmappqoecustomresp

Use this method to removes the imported HTML object.

rmappqoepolicy

Use this method to remove an AppQoE policy.

setappqoeaction_altcontentpath

Use this method to set path to the alternative content service to be used in the ACS

setappqoeaction_altcontentsvc

Use this method to set name of the alternative content service to be used in the ACS

setappqoeaction_delay

Use this method to set delay threshold, in microseconds, for requests that match the policy's rule. If the delay statistics gathered for the matching request exceed the specified delay, configured action triggered for that request, if there is no action then requests are dropped to the lowest priority level

setappqoeaction_dosaction

Use this method to set doS Action to take when vserver will be considered under DoS attack and corresponding rule matches. Mandatory if AppQoE actions are to be used for DoS attack prevention.

setappqoeaction_dostrigexpression

Use this method to set optional expression to add second level check to trigger DoS actions. Specifically used for Analytics based DoS response generation

setappqoeaction_maxconn

Use this method to set maximum number of concurrent connections that can be open for requests that matches with rule.

setappqoeaction_polqdepth

Use this method to set policy queue depth threshold value. When the policy queue size (number of requests queued for the policy binding this action is attached to) increases to the specified polqDepth value, subsequent requests are dropped to the lowest priority level.

setappqoeaction_priority

Use this method to set priority for queuing the request. If server resources are not available for a request that matches the configured rule, this option specifies a priority for queuing the request until the server resources are available again. If priority is not configured then Lowest priority will be used to queue the request.

setappqoeaction_priqdepth

Use this method to set queue depth threshold value per priorirty level. If the queue size (number of requests in the queue of that particular priorirty) on the virtual server to which this policy is bound, increases to the specified qDepth value, subsequent requests are dropped to the lowest priority level.

setappqoeparameter_avgwaitingclient

Use this method to set average number of client connections, that can sit in service waiting queue

setappqoeparameter_dosattackthresh

Use this method to set average number of client connection that can queue up on vserver level without triggering DoS mitigation module

setappqoeparameter_maxaltrespbandwidth

Use this method to set maximum bandwidth which will determine whether to send alternate content response

setappqoeparameter_sessionlife

Use this method to set time, in seconds, between the first time and the next time the AppQoE alternative content window is displayed. The alternative content window is displayed only once during a session for the same browser accessing a configured URL, so this parameter determines the length of a session.

setappqoepolicy_action

Use this method to set configured AppQoE action to trigger

setappqoepolicy_rule

Use this method to set expression or name of a named expression, against which the request is evaluated. The policy is applied if the rule evaluates to true.
