Authentication

authentication statistics.

Statistics APIs

statauthenticationpolicy

Use this method to displays authentication statistics for all advanced authentication policies, or for only the specified policy.

statauthenticationvserver

Use this method to displays statistics about the specified authentication virtual server.
If no authentication virtual server is specified, displays statistics for all authentication virtual servers that are currently configured on the NetScaler appliance.

statauthenticationsamlidppolicy

Use this method to display SAML Identity Provider (IdP) policy statistics.

statauthenticationpolicylabel

Use this method to displays statistics for the specified authentication policy label.
If no authentication policy label is specified, displays a list of all authentication policy labels.

statauthenticationpolicybinding

Use this method to display statistics of authentication policy binding(s).
