getvrid6

Use this method to get the settings of all VRID6s configured on the NetScaler appliance, or of the specified VRID6. To display the settings of all the VRID6s, run the method without any parameters. To display the settings of a particular VRID6, specify the VRID6.

Syntax



Parameters

id

Integer value that uniquely identifies a VMAC6 address.
Minimum value = 1.
Maximum value = 255.

Return Value

Returns getvrid6Result
