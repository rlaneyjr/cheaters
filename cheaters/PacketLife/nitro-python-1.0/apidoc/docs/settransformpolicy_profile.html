settransformpolicy_profile

Use this method to set name of the URL Transformation profile to use to transform requests and responses that match the policy.

Syntax



Parameters

name

Name of the policy to modify.
This is mandatory parameter.

profilename

Name of the URL Transformation profile to use to transform requests and responses that match the policy.

Return Value

Returns simpleResult
