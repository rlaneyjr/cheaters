getsslservice

Use this method to get information about SSL-specific configuration information for all SSL services, or displays detailed information about the specified SSL service.

Syntax



Parameters

servicename

Name of the SSL service for which to show detailed information.

cipherdetails

Display details of the individual ciphers bound to the SSL service.

Return Value

Returns getsslserviceResult
