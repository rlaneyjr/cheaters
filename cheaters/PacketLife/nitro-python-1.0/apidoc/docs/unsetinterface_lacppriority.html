unsetinterface_lacppriority

LACP port priority, expressed as an integer. The lower the number, the higher the priority. The NetScaler appliance limits the number of interfaces in an LA channel to sixteen.

Syntax



Parameters

id

Name of the networkinterface.
Default value = 0.

lacppriority

Indicates that lacppriority will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
