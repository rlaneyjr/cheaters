setsslparameter_dropreqwithnohostheader

Use this method to set host header check for SNI enabled sessions. If this check is enabled and the HTTP request does not contain the host header for SNI enabled sessions, the request is dropped.

Syntax



Parameters

dropreqwithnohostheader

Host header check for SNI enabled sessions. If this check is enabled and the HTTP request does not contain the host header for SNI enabled sessions, the request is dropped.
Default value = NO.
Possible Values : YES, NO.

Return Value

Returns simpleResult
