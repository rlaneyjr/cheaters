setlbsipparameters_sip503ratethreshold

Use this method to set maximum number of 503 Service Unavailable responses to generate, once every 10 milliseconds, when a SIP virtual server becomes unavailable.

Syntax



Parameters

sip503ratethreshold

Maximum number of 503 Service Unavailable responses to generate, once every 10 milliseconds, when a SIP virtual server becomes unavailable.
Default value = 100.

Return Value

Returns simpleResult
