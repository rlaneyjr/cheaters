unsetcachecontentgroup_lazydnsresolve

Remove cache contentGroup lazydnsresolve setting.

Syntax



Parameters

name

Name of the cachecontentGroup.
Default value = 0.

lazydnsresolve

Indicates that lazydnsresolve will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
