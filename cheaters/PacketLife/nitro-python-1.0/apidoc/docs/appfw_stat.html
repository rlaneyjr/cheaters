Application Firewall

application firewall statistics.

Statistics APIs

statappfw

Use this method to displays application firewall statistics.

statappfwprofile

Use this method to displays statistics for the specified application firewall profile.
If no profile is specified, displays abbreviated statistics for all profiles.

statappfwpolicy

Use this method to displays statistics for the specified application firewall policy.
If no application firewall policy is specified, displays abbreviated statistics for all application firewall policies.

statappfwpolicylabel

Use this method to displays statistics for the specified application firewall policy label.
If no application firewall policy label is specified, displays abbreviated statistics for all application firewall policy labels.

statappfwpolicybinding

Use this method to display statistics of advanced application firewall policy binding(s).
