getgslbsite

Use this method to get the parameters of all the GSLB sites configured on the appliance, or the parameters of the specified GSLB site.

Syntax



Parameters

sitename

Name of the GSLB site. If you specify a site name, details of all the site's constituent services are also displayed.

Return Value

Returns getgslbsiteResult
