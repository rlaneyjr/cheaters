addvpnnexthopserver

Use this method to enables a NetScaler Gateway appliance in the first DMZ to communicate with one or more NetScaler Gateway appliances in the second DMZ.

Syntax



Parameters

name

Name for the NetScaler Gateway appliance in the first DMZ.
This is mandatory parameter.

nexthopip

IP address or FQDN of the NetScaler Gateway proxy in the second DMZ.
This is mandatory parameter.

nexthopport

Port number of the NetScaler Gateway proxy in the second DMZ.
This is mandatory parameter.
Minimum value = 1.
Maximum value = 65535.

secure

Use of a secure port, such as 443, for the double-hop configuration.
Default value = OFF.

Return Value

Returns simpleResult
