getroute6

Use this method to get configuration and state information of all IPv6 routes in the NetScaler appliance's routing table, or of the specified IPv6 route.

Syntax



Parameters

network

IPv6 network address of the route entry for which to display details.

gateway

Any gateway of the route entry for which to display details.
Default value = 0.

vlan

Integer that uniquely identifies the VLAN defined for this route.
Default value = 0.
Minimum value = 0.
Maximum value = 4094.

Return Value

Returns getroute6Result
