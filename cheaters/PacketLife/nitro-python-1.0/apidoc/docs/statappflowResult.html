statappflowResult Structure Definition

The statappflowResult structure defines the return type for statappflow API.

Syntax



Members

rc

If the method succeeds, rc is 0 else rc > 0. Values above 0x8000 indicate Warnings.

message

If the method succeeds, message is NULL else message contains Error/Warning message.

appflowList

Pointer to the appflow List.


appflowstats Structure Definition

The appflowstats structure defines the actual return type values for statappflow API.

Syntax



Members

clearstats

Clear the statsistics / counters

appflowtotaltxoctets

The total number of AppFlow (IPFIX) octets that the NetScaler has transmitted.

appflowtxoctetsrate

The Rate of AppFlow (IPFIX) octets that the NetScaler has transmitted.

appflowtotalflows

The total number of AppFlow (IPFIX) flows that the NetScaler has transmitted.

appflowflowsrate

The Rate of AppFlow (IPFIX) flows that the NetScaler has transmitted.

appflowtotaltxmessagess

The total number of AppFlow (IPFIX) messages that the NetScaler has transmitted.

appflowtxmessagessrate

The Rate of AppFlow (IPFIX) messages that the NetScaler has transmitted.

appflowtotalignoctets

The total number of octets that the NetScaler has ignored for AppFlow (IPFIX).

appflowignoctetsrate

The Rate of octets that the NetScaler has ignored for AppFlow (IPFIX).

appflowtotalignpacketss

The total number of packets that the NetScaler has ignored for AppFlow (IPFIX).

appflowignpacketssrate

The Rate of packets that the NetScaler has ignored for AppFlow (IPFIX).

appflowtotalnotxoctets

The total number of AppFlow (IPFIX) octets that the NetScaler has not transmitted.

appflownotxoctetsrate

The Rate of AppFlow (IPFIX) octets that the NetScaler has not transmitted.

appflowtotalnotxflows

The total number of AppFlow (IPFIX) flows that the NetScaler has not transmitted.

appflownotxflowsrate

The Rate of AppFlow (IPFIX) flows that the NetScaler has not transmitted.

appflowtotalnotxpackets

The total number of AppFlow (IPFIX) packets that the NetScaler has not transmitted.

appflownotxpacketsrate

The Rate of AppFlow (IPFIX) packets that the NetScaler has not transmitted.
