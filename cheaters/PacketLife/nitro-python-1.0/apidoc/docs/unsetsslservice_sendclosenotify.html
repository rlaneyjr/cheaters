unsetsslservice_sendclosenotify

Remove ssl service sendclosenotify setting.

Syntax



Parameters

servicename

Name of the sslservice.
Default value = 0.

sendclosenotify

Indicates that sendclosenotify will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
