Appflow

appflow configuration.

Configuration APIs

addappflowaction

Use this method to creates an AppFlow action. The action can be associated with an AppFlow policy by using the add appflow policy method.

addappflowcollector

Use this method to adds a new AppFlow collector. A collector receives the flow records generated by the NetScaler appliance.
You can add only four AppFlow collectors to the NetScaler appliance.

addappflowpolicy

Use this method to adds an Appflow policy. The policy specifies the rule based on which the traffic is evaluated, and the action to be taken if the rule returns "TRUE".

addappflowpolicylabel

Use this method to creates a user-defined AppFlow policy label. You can bind AppFlow policies to the AppFlow policy label.

bindappflowglobal_policy

Use this method to bind policy to appflow global.

bindappflowpolicylabel_policy

Use this method to bind policy to appflow policylabel.

getappflowaction

Use this method to get information about AppFlow action(s), or about the specified AppFlow action.

getappflowcollector

Use this method to get information about all configured AppFlow collectors, or about the specified collector.

getappflowglobal

Use this method to get the AppFlow global bind points and the number of policies bound to each global bind point, or more detailed information about the specified bind point.

getappflowparam

Use this method to get AppFlow parameters.

getappflowpolicy

Use this method to get information about all configured AppFlow policies, or detailed information about the specified policy.

getappflowpolicylabel

Use this method to get information about all AppFlow policy labels, or detailed information about the specified policy label.

renameappflowaction

Use this method to renames an AppFlow action.

renameappflowcollector

Use this method to renames an AppFlow collector.

renameappflowpolicy

Use this method to renames an AppFlow policy.

renameappflowpolicylabel

Use this method to renames an AppFlow policy label.

rmappflowaction

Use this method to removes a configured AppFlow action. You cannot remove an action that is associated with an AppFlow policy.

rmappflowcollector

Use this method to removes an AppFlow collector. You cannot remove a collector if it is associated with an AppFlow action.

rmappflowpolicy

Use this method to removes an AppFlow policy. (Cannot remove a policy that is bound to a policy label.)

rmappflowpolicylabel

Use this method to removes an AppFlow policy label.

setappflowaction_clientsidemeasurements

Use this method to set on enabling this option, the NetScaler will collect the time required to load and render the mainpage on the client.

setappflowaction_collectors

Use this method to set name(s) of collector(s) to be associated with the AppFlow action.

setappflowaction_comment

Use this method to set any comments about this action. In the CLI, if including spaces between words, enclose the comment in quotation marks. (The quotation marks are not required in the configuration utility.)

setappflowparam_aaauser

Use this method to set enable AppFlow AAA Username logging.

setappflowparam_appnamerefresh

Use this method to set interval, in seconds, at which to send Appnames to the configured collectors. Appname refers to the name of an entity (virtual server, service, or service group) in the NetScaler appliance.

setappflowparam_clienttrafficonly

Use this method to set generate AppFlow records for only the traffic from the client.

setappflowparam_connectionchaining

Use this method to set enable connection chaining so that the client server flows of a connection are linked. Also the connection chain ID is propagated across NetScalers, so that in a multi-hop environment the flows belonging to the same logical connection are linked. This id is also logged as part of appflow record

setappflowparam_flowrecordinterval

Use this method to set interval, in seconds, at which to send flow records to the configured collectors.

setappflowparam_httpauthorization

Use this method to set include the HTTP Authorization header information.

setappflowparam_httpcontenttype

Use this method to set include the HTTP Content-Type header sent from the server to the client to determine the type of the content sent.

setappflowparam_httpcookie

Use this method to set include the cookie that was in the HTTP request the appliance received from the client.

setappflowparam_httpdomain

Use this method to set include the http domain request to be exported.

setappflowparam_httphost

Use this method to set include the host identified in the HTTP request that the appliance received from the client.

setappflowparam_httplocation

Use this method to set include the HTTP location headers returned from the HTTP responses.

setappflowparam_httpmethod

Use this method to set include the method that was specified in the HTTP request that the appliance received from the client.

setappflowparam_httpreferer

Use this method to set include the web page that was last visited by the client.

setappflowparam_httpsetcookie

Use this method to set include the Set-cookie header sent from the server to the client in response to a HTTP request.

setappflowparam_httpsetcookie2

Use this method to set include the Set-cookie header sent from the server to the client in response to a HTTP request.

setappflowparam_httpurl

Use this method to set include the http URL that the NetScaler appliance received from the client.

setappflowparam_httpuseragent

Use this method to set include the client application through which the HTTP request was received by the NetScaler appliance.

setappflowparam_httpvia

Use this method to set include the httpVia header which contains the IP address of proxy server through which the client accessed the server.

setappflowparam_httpxforwardedfor

Use this method to set include the httpXForwardedFor header, which contains the original IP Address of the client using a proxy server to access the server.

setappflowparam_skipcacheredirectionhttptransaction

Use this method to set skip Cache http transaction. This HTTP transaction is specific to Cache Redirection module. In Case of Cache Miss there will be another HTTP transaction initiated by the cache server.

setappflowparam_templaterefresh

Use this method to set refresh interval, in seconds, at which to export the template data. Because data transmission is in UDP, the templates must be resent at regular intervals.

setappflowparam_udppmtu

Use this method to set mTU, in bytes, for IPFIX UDP packets.

setappflowpolicy_action

Use this method to set name of the action to be associated with this policy.

setappflowpolicy_comment

Use this method to set any comments about this policy.

setappflowpolicy_rule

Use this method to set expression or other value against which the traffic is evaluated. Must be a Boolean, default syntax expression. Note:
Maximum length of a string literal in the expression is 255 characters. A longer string can be split into smaller strings of up to 255 characters each, and the smaller strings concatenated with the + operator. For example, you can create a 500-character string as follows: '"" + ""'

The following requirements apply only to the NetScaler CLI:
* If the expression includes one or more spaces, enclose the entire expression in double quotation marks.
* If the expression itself includes double quotation marks, escape the quotations by using the \ character.
* Alternatively, you can use single quotation marks to enclose the rule, in which case you do not have to escape the double quotation marks.
