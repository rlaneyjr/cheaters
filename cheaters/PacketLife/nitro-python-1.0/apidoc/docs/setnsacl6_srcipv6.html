setnsacl6_srcipv6

Use this method to set iP address or range of IP addresses to match against the source IP address of an incoming IPv6 packet. In the method line interface, separate the range with a hyphen and enclose within brackets.

Syntax



Parameters

acl6name

Name of the ACL6 rule whose parameters you want to modify.
This is mandatory parameter.

srcipv6

IP address or range of IP addresses to match against the source IP address of an incoming IPv6 packet. In the method line interface, separate the range with a hyphen and enclose within brackets.

operatoR

Logical operator.
Possible Values : =, !=, EQ, NEQ.

srcipv6val

Source IPv6 address (range).

Return Value

Returns simpleResult
