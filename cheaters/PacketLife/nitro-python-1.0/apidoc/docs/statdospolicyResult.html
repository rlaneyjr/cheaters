statdospolicyResult Structure Definition

The statdospolicyResult structure defines the return type for statdospolicy API.

Syntax



Members

rc

If the method succeeds, rc is 0 else rc > 0. Values above 0x8000 indicate Warnings.

message

If the method succeeds, message is NULL else message contains Error/Warning message.

dospolicyList

Pointer to the dospolicy List.


dospolicystats Structure Definition

The dospolicystats structure defines the actual return type values for statdospolicy API.

Syntax



Members

name

The name of the DoS protection policy whose statistics must be displayed. If a name is not provided, statistics of all the DoS protection policies are displayed.

clearstats

Clear the statsistics / counters

doscurrentcltdetectrate

Current ratio of JavaScript send rate to the server response rate (Client detect rate)

dosphysicalserviceip

IP address of the service to which this policy is bound.

dosphysicalserviceport

Port address of the service to which this policy is bound.

doscurrentqueuesize

Current queue size of the server to which this policy is bound.

doscurrentqueuesizerate

Current queue size of the server to which this policy is bound.

dostotjssent

Total number of DoS JavaScript transactions performed for this policy.

dosjssentrate

Rate of DoS JavaScript transactions performed for this policy.

dostotjsrefused

Number of times the DoS JavaScript was not sent because the set JavaScript rate was not met for this policy.

dosjsrefusedrate

Rate of times the DoS JavaScript was not sent because the set JavaScript rate was not met for this policy.

dostotvalidclients

Total number of valid DoS cookies received for this policy.

dosvalidclientsrate

Rate of valid DoS cookies received for this policy.

dostotjsbytessent

Total number of DoS JavaScript bytes sent for this policy.

dosjsbytessentrate

Rate of DoS JavaScript bytes sent for this policy.

dostotnongetpostrequests

Number of non-GET and non-POST requests for which DOS JavaScript was sent.

dosnongetpostrequestsrate

Rate of non-GET and non-POST requests for which DOS JavaScript was sent.

doscurrentjsrate

Current rate at which JavaScript is being sent in response to client requests.

doscurrentjsraterate

Current rate at which JavaScript is being sent in response to client requests.

doscurserverresprate

Current rate at which the server to which this policy is bound is responding.

doscurserverrespraterate

Current rate at which the server to which this policy is bound is responding.
