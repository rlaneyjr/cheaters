settransformaction_cookiedomainfrom

Use this method to set pattern that matches the domain to be transformed in Set-Cookie headers.

Syntax



Parameters

name

Name of the URL Transformation action to modify.
This is mandatory parameter.

cookiedomainfrom

Pattern that matches the domain to be transformed in Set-Cookie headers.

Return Value
