AAA

authentication, authorization, and accounting service configuration.

Configuration APIs

addaaagroup

Use this method to creates a AAA group and verifies the configuration to ensure that it is correct.

addaaakcdaccount

Use this method to add a Kerberos constrained delegation account.

addaaapreauthenticationaction

Use this method to adds an action (profile) for endpoint analysis (EPA) clients before authentication.

addaaapreauthenticationpolicy

Use this method to adds a preauthentication policy. The policy defines expressions to be evaluated by the endpoint analysis (EPA) tool.

addaaauser

Use this method to adds a local AAA user account and verifies the configuration to ensure that it is correct.

bindaaaglobal_policy

Use this method to bind policy to aaa global.

bindaaaglobal_windowsprofile

Use this method to bind windowsprofile to aaa global.

bindaaagroup_intranetapplication

Use this method to bind intranetapplication to aaa group.

bindaaagroup_intranetip

Use this method to bind intranetip to aaa group.

bindaaagroup_policy

Use this method to bind policy to aaa group.

bindaaagroup_url

Use this method to bind url to aaa group.

bindaaagroup_user

Use this method to bind user to aaa group.

bindaaauser_intranetapplication

Use this method to bind intranetapplication to aaa user.

bindaaauser_intranetip

Use this method to bind intranetip to aaa user.

bindaaauser_policy

Use this method to bind policy to aaa user.

bindaaauser_url

Use this method to bind url to aaa user.

getaaacertparams

Use this method to get the current client certificate configuration on the NetScaler appliance.

getaaaglobal

Use this method to get a list of policies that are currently bound to Global on the NetScaler appliance.

getaaagroup

Use this method to get the current configuration of a AAA group.

getaaakcdaccount

Use this method to get KCD accounts.

getaaaldapparams

Use this method to get the current LDAP configuration on the NetScaler appliance.

getaaaparameter

Use this method to get the current AAA global configuration.

getaaapreauthenticationaction

Use this method to get details of the specified preauthentication action.

getaaapreauthenticationparameter

Use this method to get the current preauthentication configuration.

getaaapreauthenticationpolicy

Use this method to get the properties of either the specified preauthentication policy or (if none is specified) a list of all configured preauthentication policies.

getaaaradiusparams

Use this method to get the current RADIUS configuration on the NetScaler appliance.

getaaasession

Use this method to get all AAA-TM/VPN connections that are bound to the specified user, group, IP address, or IP range.

getaaatacacsparams

Use this method to get the NetScaler appliance’s current AAA TACACS+ configuration.

getaaauser

Use this method to get the current configuration of a AAA user account.

killaaasession

Use this method to terminates the specified AAA-TM/VPN session.

rmaaagroup

Use this method to removes the specified AAA group.

rmaaakcdaccount

Use this method to remove the KCD account.

rmaaapreauthenticationaction

Use this method to removes a preauthentication action.
NOTE: A preauthentication action cannot be removed if it is bound to a policy.

rmaaapreauthenticationpolicy

Use this method to removes the specified preauthentication policy.

rmaaauser

Use this method to removes a local AAA user account and the associated configuration.

setaaacertparams_defaultauthenticationgroup

Use this method to set this is the default group that is chosen when the authentication succeeds in addition to extracted groups.

setaaacertparams_groupnamefield

Use this method to set client certificate field that specifies the group, in the format :.

setaaacertparams_usernamefield

Use this method to set client certificate field that contains the username, in the format :.

setaaakcdaccount_cacert

Use this method to set cA Cert for UserCert or when doing PKINIT backchannel.

setaaakcdaccount_delegateduser

Use this method to set username that can perform kerberos constrained delegation.

setaaakcdaccount_enterpriserealm

Use this method to set enterprise Realm of the user. This should be given only in certain KDC deployments where KDC expects Enterprise username instead of Principal Name

setaaakcdaccount_kcdpassword

Use this method to set password for Delegated User.

setaaakcdaccount_keytab

Use this method to set the path to the keytab file. If specified other parameters in this method need not be given

setaaakcdaccount_realmstr

Use this method to set kerberos Realm.

setaaakcdaccount_servicespn

Use this method to set service SPN. When specified, this will be used to fetch kerberos tickets. If not specified, Netscaler will construct SPN using service fqdn

setaaakcdaccount_usercert

Use this method to set sSL Cert (including private key) for Delegated User.

setaaakcdaccount_userrealm

Use this method to set realm of the user

setaaaldapparams_authtimeout

Use this method to set maximum number of seconds that the NetScaler appliance waits for a response from the LDAP server.

setaaaldapparams_defaultauthenticationgroup

Use this method to set this is the default group that is chosen when the authentication succeeds in addition to extracted groups.

setaaaldapparams_groupattr

Use this method to set attribute name used for group extraction from the LDAP server.

setaaaldapparams_ldapbase

Use this method to set base (the server and location) from which LDAP search methods should start.
If the LDAP server is running locally, the default value of base is dc=netscaler, dc=com.

setaaaldapparams_ldapbinddn

Use this method to set complete distinguished name (DN) string used for binding to the LDAP server.

setaaaldapparams_ldapbinddnpassword

Use this method to set password for binding to the LDAP server.

setaaaldapparams_ldaplogin

Use this method to set name attribute that the NetScaler appliance uses to query the external LDAP server or an Active Directory.

setaaaldapparams_nestedgroupextraction

Use this method to set queries the external LDAP server to determine whether the specified group belongs to another group.

setaaaldapparams_passwdchange

Use this method to set accept password change requests.

setaaaldapparams_searchfilter

Use this method to set string to be combined with the default LDAP user search string to form the value to use when executing an LDAP search.
For example, the following values:
vpnallowed=true,
ldaploginame=""samaccount""
when combined with the user-supplied username ""bob"", yield the following LDAP search string:
""(&(vpnallowed=true)(samaccount=bob)""

setaaaldapparams_sectype

Use this method to set type of security used for communications between the NetScaler appliance and the LDAP server. For the PLAINTEXT setting, no encryption is required.

setaaaldapparams_serverip

Use this method to set iP address of your LDAP server.

setaaaldapparams_serverport

Use this method to set port number on which the LDAP server listens for connections.

setaaaldapparams_ssonameattribute

Use this method to set attribute used by the NetScaler appliance to query an external LDAP server or Active Directory for an alternative username.
This alternative username is then used for single sign-on (SSO).

setaaaldapparams_subattribute

Use this method to set subattribute name used for group extraction from the LDAP server.

setaaaldapparams_svrtype

Use this method to set the type of LDAP server.

setaaaparameter_aaadnatip

Use this method to set source IP address to use for traffic that is sent to the authentication server.

setaaaparameter_defaultauthtype

Use this method to set the default authentication server type.

setaaaparameter_enableenhancedauthfeedback

Use this method to set enhanced auth feedback provides more information to the end user about the reason for an authentication failure. The default value is set to NO.

setaaaparameter_enablesessionstickiness

Use this method to set enables/Disables stickiness to authentication servers

setaaaparameter_enablestaticpagecaching

Use this method to set the default state of VPN Static Page caching. If nothing is specified, the default value is set to YES.

setaaaparameter_maxaaausers

Use this method to set maximum number of concurrent users allowed to log on to VPN simultaneously.

setaaaparameter_maxloginattempts

Use this method to set maximum Number of login Attempts

setaaapreauthenticationaction_deletefiles

Use this method to set string specifying the path(s) and name(s) of the files to be deleted by the endpoint analysis (EPA) tool.

setaaapreauthenticationaction_killprocess

Use this method to set string specifying the name of a process to be terminated by the endpoint analysis (EPA) tool.

setaaapreauthenticationaction_preauthenticationaction

Use this method to set allow or deny logon after endpoint analysis (EPA) results.

setaaapreauthenticationparameter_deletefiles

Use this method to set string specifying the path(s) to and name(s) of the files to be deleted by the EPA tool, as a string of between 1 and 1023 characters.

setaaapreauthenticationparameter_killprocess

Use this method to set string specifying the name of a process to be terminated by the EPA tool.

setaaapreauthenticationparameter_preauthenticationaction

Use this method to set deny or allow login on the basis of end point analysis results.

setaaapreauthenticationparameter_rule

Use this method to set name of the NetScaler named rule, or a default syntax expression, to be evaluated by the EPA tool.

setaaapreauthenticationpolicy_reqaction

Use this method to set name of the action that the policy is to invoke when a connection matches the policy.

setaaapreauthenticationpolicy_rule

Use this method to set the new rule to be associated with the policy.

setaaaradiusparams_accounting

Use this method to set configure the RADIUS server state to accept or refuse accounting messages.

setaaaradiusparams_authtimeout

Use this method to set maximum number of seconds that the NetScaler appliance waits for a response from the RADIUS server.

setaaaradiusparams_callingstationid

Use this method to set send Calling-Station-ID of the client to the RADIUS server. IP Address of the client is sent as its Calling-Station-ID.

setaaaradiusparams_defaultauthenticationgroup

Use this method to set this is the default group that is chosen when the authentication succeeds in addition to extracted groups.

setaaaradiusparams_ipattributetype

Use this method to set iP attribute type in the RADIUS response.

setaaaradiusparams_ipvendorid

Use this method to set vendor ID attribute in the RADIUS response.
If the attribute is not vendor-encoded, it is set to 0.

setaaaradiusparams_passencoding

Use this method to set enable password encoding in RADIUS packets that the NetScaler appliance sends to the RADIUS server.

setaaaradiusparams_pwdvendorid

Use this method to set vendor ID of the password in the RADIUS response. Used to extract the user password.

setaaaradiusparams_radattributetype

Use this method to set attribute type for RADIUS group extraction.

setaaaradiusparams_radgroupseparator

Use this method to set group separator string that delimits group names within a RADIUS attribute for RADIUS group extraction.

setaaaradiusparams_radgroupsprefix

Use this method to set prefix string that precedes group names within a RADIUS attribute for RADIUS group extraction.

setaaaradiusparams_radkey

Use this method to set the key shared between the RADIUS server and clients.
Required for allowing the NetScaler appliance to communicate with the RADIUS server.
This is mandatory parameter.

setaaaradiusparams_radnasid

Use this method to set send the Network Access Server ID (NASID) for your NetScaler appliance to the RADIUS server as the nasid part of the Radius protocol.

setaaaradiusparams_radnasip

Use this method to set send the NetScaler IP (NSIP) address to the RADIUS server as the Network Access Server IP (NASIP) part of the Radius protocol.

setaaaradiusparams_radvendorid

Use this method to set vendor ID for RADIUS group extraction.

setaaaradiusparams_serverip

Use this method to set iP address of your RADIUS server.

setaaaradiusparams_serverport

Use this method to set port number on which the RADIUS server listens for connections.

setaaatacacsparams_accounting

Use this method to set send accounting messages to the TACACS+ server.

setaaatacacsparams_auditfailedcmds

Use this method to set the option for sending accounting messages to the TACACS+ server.

setaaatacacsparams_authorization

Use this method to set use streaming authorization on the TACACS+ server.

setaaatacacsparams_authtimeout

Use this method to set maximum number of seconds that the NetScaler appliance waits for a response from the TACACS+ server.

setaaatacacsparams_defaultauthenticationgroup

Use this method to set this is the default group that is chosen when the authentication succeeds in addition to extracted groups.

setaaatacacsparams_serverip

Use this method to set iP address of your TACACS+ server.

setaaatacacsparams_serverport

Use this method to set port number on which the TACACS+ server listens for connections.

setaaatacacsparams_tacacssecret

Use this method to set key shared between the TACACS+ server and clients. Required for allowing the NetScaler appliance to communicate with the TACACS+ server.

setaaauser_password

Use this method to set password with which the user logs on. Required for any user account that does not exist on an external authentication server.
If you are not using an external authentication server, all user accounts must have a password. If you are using an external authentication server, you must provide a password for local user accounts that do not exist on the authentication server.
This is mandatory parameter.
