statpqpolicyResult Structure Definition

The statpqpolicyResult structure defines the return type for statpqpolicy API.

Syntax



Members

rc

If the method succeeds, rc is 0 else rc > 0. Values above 0x8000 indicate Warnings.

message

If the method succeeds, message is NULL else message contains Error/Warning message.

pqpolicyList

Pointer to the pqpolicy List.


pqpolicystats Structure Definition

The pqpolicystats structure defines the actual return type values for statpqpolicy API.

Syntax



Members

policyname

Name of the priority queuing policy whose statistics must be displayed. If a name is not provided, statistics of all priority queuing policies are shown.

clearstats

Clear the statsistics / counters

Amount of time spent by priority queuing clients waiting in the priority queue.

Amount of time spent by priority queuing clients waiting in the priority queue.
