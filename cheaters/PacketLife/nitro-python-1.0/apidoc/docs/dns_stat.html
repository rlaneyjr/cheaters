Domain Name Service

domain name service(dns) statistics.

Statistics APIs

statdns

Use this method to displays DNS statistics.

statdnsrecords

Use this method to displays statistics for the specified DNS record or query type. If a DNS record or query type is not specified, statistics for all record and query types are shown.

statdnspolicylabel

Use this method to display statistics of dns policylabel(s).

statdnspolicybinding

Use this method to display statistics of dns policy binding(s).
