Filter

filter configuration. protection from malicious attacks can be achieved for web sites at the content level with the system's content filtering feature. when this feature is enabled, the system inspects every incoming request with the user-configured rules based on any http url or headers.

Configuration APIs

addfilteraction

Use this method to creates a content filtering action. This action can be associated with a content filtering policy that is created with the add filter policy method.
Note: The following content filtering actions are available by default:
* RESET - Sends a TCP reset for the HTTP requests.
* DROP - Drops the HTTP requests silently, without sending a TCP FIN for closing the connection.

addfilterhtmlinjectionvariable

Use this method to creates an HTML injection variable.

addfilterpolicy

Use this method to creates a content filtering policy.

bindfilterglobal_policy

Use this method to bind policy to filter global.

getfilteraction

Use this method to get information about available filtering actions.

getfilterglobal

Use this method to get the globally activated filter policies.

getfilterhtmlinjectionparameter

Use this method to get the HTML injection parameters.

getfilterhtmlinjectionvariable

Use this method to get information about HTML injection variables.

getfilterpolicy

Use this method to get information about the filter policies.

getfilterpostbodyinjection

Use this method to get the name of the file used for postbody injection.

getfilterprebodyinjection

Use this method to get the name of the file used for prebody injection.

rmfilteraction

Use this method to removes a content filtering action.

rmfilterhtmlinjectionvariable

Use this method to removes an HTML injection variable.

rmfilterpolicy

Use this method to removes a filter policy.

setfilteraction_page

Use this method to set hTML page to return for HTTP requests (For use with the ERRORCODE qualifier).

setfilteraction_respcode

Use this method to set response code to be returned for HTTP requests (for use with the ERRORCODE qualifier).

setfilteraction_service

Use this method to set service to which to forward HTTP requests. Required if the qualifier is FORWARD.

setfilteraction_value

Use this method to set string containing the header_name and header_value. If the qualifier is ADD, specify :. If the qualifier is CORRUPT, specify only the header_name

setfilterhtmlinjectionparameter_frequency

Use this method to set for a frequency of x, HTML injection is done at least once per x milliseconds.

setfilterhtmlinjectionparameter_htmlsearchlen

Use this method to set number of characters, in the HTTP body, in which to search for the tag if strict mode is set.

setfilterhtmlinjectionparameter_rate

Use this method to set for a rate of x, HTML injection is done for 1 out of x policy matches.

setfilterhtmlinjectionparameter_strict

Use this method to set searching for tag. If this parameter is enabled, HTML injection does not insert the prebody or postbody content unless the tag is found.

setfilterhtmlinjectionvariable_value

Use this method to set value to be assigned to the new variable.

setfilterpolicy_reqaction

Use this method to set name of the action to be performed on requests that match the policy. Cannot be specified if the rule includes condition to be evaluated for responses.

setfilterpolicy_resaction

Use this method to set the action to be performed on the response. The string value can be a filter action created filter action or a built-in action.

setfilterpolicy_rule

Use this method to set netScaler classic expression specifying the type of connections that match this policy.

setfilterpostbodyinjection_postbody

Use this method to set name of file whose contents are to be inserted after the response body.
This is mandatory parameter.

setfilterprebodyinjection_prebody

Use this method to set name of file whose contents are to be inserted before the response body.
This is mandatory parameter.

unbindfilterglobal_policy

Use this method to unbind policy from filter global.

unsetfilteraction_page

Remove filter action page setting.

unsetfilterhtmlinjectionparameter_frequency

Remove filter htmlinjectionparameter frequency setting.

unsetfilterhtmlinjectionparameter_htmlsearchlen

Remove filter htmlinjectionparameter htmlsearchlen setting.

unsetfilterhtmlinjectionparameter_rate

Remove filter htmlinjectionparameter rate setting.

unsetfilterhtmlinjectionparameter_strict

Remove filter htmlinjectionparameter strict setting.

unsetfilterhtmlinjectionvariable_value

Remove filter htmlinjectionvariable value setting.

unsetfilterpostbodyinjection_postbody

Remove filter postbodyInjection postbody setting.

unsetfilterprebodyinjection_prebody

Remove filter prebodyInjection prebody setting.
