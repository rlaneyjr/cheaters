NS

system/global level statistics.

Statistics APIs

statns

Use this method to displays generic statistics of the NetScaler appliance.

statnslimitidentifier

Use this method to display statistics of a identifier.

statnsacl

Use this method to displays statistics related to the extended ACL rules. To display statistics of all the extended ACL rules, run the method without any parameters. To display statistics of a particular extended ACL rule, specify the name of the extended ACL rule.

statnsacl6

Use this method to displays statistics related to the ACL6 rules. To display statistics of all the ACL6 rules, run the method without any parameters. To display statistics of a particular ACL6 rule, specify the name of the ACL6 rule.

statnssimpleacl

Use this method to displays statistics related to the simple ACL rules.

statnssimpleacl6

Use this method to displays statistics related to the simple ACL6 rules.

statnspbr

Use this method to displays statistics related to the PBRs. To display statistics of all the PBRs, run the method without any parameters. To display statistics of a particular PBR, specify the name of the PBR.

statnslimitsessions

Use this method to display statistics of a individual member.

statnsmemory

Use this method to displays memory statistics of NetScaler features.

statnspbr6

Use this method to displays statistics related to the PBR6s. To display statistics of all the PBR6s, run the method without any parameters. To display statistics of a particular PBR6, specify the name of the PBR6.

statnstrafficdomain

Use this method to display statistics for Traffic Domains(s).
