setvpnsessionaction_dnsvserver

Use this method to set name of the DNS virtual server for the user session.

Syntax



Parameters

name

The name of the vpn session action.
This is mandatory parameter.

dnsvservername

Name of the DNS virtual server for the user session.

Return Value

Returns simpleResult
