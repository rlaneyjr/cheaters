setauthenticationradiusaction_ipattributetype

Use this method to set remote IP address attribute type in a RADIUS response.

Syntax



Parameters

name

Name of the RADIUS profile.
This is mandatory parameter.

ipattributetype

Remote IP address attribute type in a RADIUS response.
Minimum value = 1.

Return Value

Returns simpleResult
