setdnssoarec_refresh

Use this method to set time, in seconds, for which a secondary server must wait between successive checks on the value of the serial number.

Syntax



Parameters

domain

Domain of the SOA record to be modified.
This is mandatory parameter.

refresh

Time, in seconds, for which a secondary server must wait between successive checks on the value of the serial number.
Default value = 3600.
Maximum value = 0xFFFFFFFE.

Return Value

Returns simpleResult
