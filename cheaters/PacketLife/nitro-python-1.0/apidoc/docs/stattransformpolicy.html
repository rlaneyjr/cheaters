stattransformpolicy

Use this method to displays statistics for the specified URL Transformation policy.
If no policy name is specified, displays abbreviated statistics for all URL Transformation policies currently configured on the NetScaler appliance.

Syntax



Parameters

name

Name of the policy.

clearstats

Clear the statsistics / counters
Possible Values : basic, full.

Return Value

Returns stattransformpolicyResult
