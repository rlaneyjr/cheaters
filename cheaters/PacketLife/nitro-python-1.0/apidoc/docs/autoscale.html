autoscale configuration.

Configuration APIs

addautoscaleaction

Use this method to create a AutoScale action.

addautoscalepolicy

Use this method to create a autoscale policy.

addautoscaleprofile

Use this method to create a AutoScale policy.

getautoscaleaction

Use this method to get the autoscale actions.

getautoscalepolicy

Use this method to get the autoscale policies.

getautoscaleprofile

Use this method to get the autoscale profile.

renameautoscalepolicy

Use this method to rename a autoscale policy.

rmautoscaleaction

Use this method to remove a AutoScale action.

rmautoscalepolicy

Use this method to remove a autoscale policy.

rmautoscaleprofile

Use this method to remove a AutoScale policy.

setautoscaleaction_parameters

Use this method to set parameters to use in the action

setautoscaleaction_profile

Use this method to set autoScale profile name.

setautoscaleaction_quiettime

Use this method to set time in seconds no other policy is evaluated or action is taken

setautoscaleaction_vmdestroygraceperiod

Use this method to set time in minutes a VM is kept in inactive state before destroying

setautoscaleaction_vserver

Use this method to set name of the vserver on which autoscale action has to be taken.

setautoscalepolicy_action

Use this method to set the autoscale profile associated with the policy.

setautoscalepolicy_comment

Use this method to set comments associated with this autoscale policy.

setautoscalepolicy_logaction

Use this method to set the log action associated with the autoscale policy

setautoscalepolicy_rule

Use this method to set the rule associated with the policy.

setautoscaleprofile_apikey

Use this method to set api key for authentication with service

setautoscaleprofile_sharedsecret

Use this method to set shared secret for authentication with service

setautoscaleprofile_url

Use this method to set uRL providing the service

unsetautoscaleaction_quiettime

Remove autoscale action quiettime setting.

unsetautoscaleaction_vmdestroygraceperiod

Remove autoscale action vmdestroygraceperiod setting.

unsetautoscalepolicy_action

Remove autoscale policy action setting.

unsetautoscalepolicy_comment

Remove autoscale policy comment setting.

unsetautoscalepolicy_logaction

Remove autoscale policy logaction setting.

unsetautoscalepolicy_rule

Remove autoscale policy rule setting.
