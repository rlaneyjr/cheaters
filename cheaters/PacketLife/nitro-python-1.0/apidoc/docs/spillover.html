Spillover

spillover policies and actions.

Configuration APIs

addspilloveraction

Use this method to creating spillover action

addspilloverpolicy

Use this method to add a spillover policy. SPILLOVER policies that can be added are based on vserver expressions.

getspilloveraction

Use this method to get spillover actions

getspilloverpolicy

Use this method to get the policy-related information.

renamespilloveraction

Use this method to renames a spillover action.

renamespilloverpolicy

Use this method to renames a spillover policy.

rmspilloveraction

Use this method to removes a spillover policy.

rmspilloverpolicy

Use this method to removes a spillover policy.

setspilloverpolicy_action

Use this method to set action for the spillover policy. Action is created using add spillover action method

setspilloverpolicy_comment

Use this method to set any comments that you might want to associate with the spillover policy.

setspilloverpolicy_rule

Use this method to set expression to be used by the spillover policy.

unsetspilloverpolicy_comment

Remove spillover policy comment setting.
