virtual private network statistics.

Statistics APIs

statvpn

Use this method to displays the statistics for NetScaler Gateway usage. Displays event information, such as the event that generated the message, a time stamp, the message type, and predefined log levels and message information.

statvpnvserver

Use this method to displays statistics for all NetScaler Gateway virtual servers, or displays detailed statistics for the specified NetScaler Gateway virtual server.
