unbindvlan_ifnum

Interface to unbind from the VLAN, specified in slot/port notation (for example, 1/3).

Syntax



Parameters

id

The virtual LAN (VLAN) id.
This is mandatory parameter.
Minimum value = 1.
Maximum value = 4094.

ifnum

Interface to unbind from the VLAN, specified in slot/port notation (for example, 1/3).

tagged

The 802.1q tagged interface.

Return Value

Returns simpleResult
