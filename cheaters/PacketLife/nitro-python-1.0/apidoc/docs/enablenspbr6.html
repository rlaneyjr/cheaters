enablenspbr6

Use this method to enables a PBR6. To commit this operation, you must apply the PBR6s.After you apply the PBR6s, the NetScaler appliance compares outgoing packets to the enabled PBR6.

Syntax



Parameters

name

Name of PBR6 that you want to enable.
This is mandatory parameter.

Return Value

Returns simpleResult
