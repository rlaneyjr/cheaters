setdospolicy_qdepth

Use this method to set queue depth. The queue size (the number of outstanding service requests on the system) before DoS protection is activated on the service to which the DoS protection policy is bound.

Syntax



Parameters

name

Name of the DoS protection policy to be modified.
This is mandatory parameter.

qdepth

Queue depth. The queue size (the number of outstanding service requests on the system) before DoS protection is activated on the service to which the DoS protection policy is bound.
Minimum value = 21.

Return Value

Returns simpleResult
