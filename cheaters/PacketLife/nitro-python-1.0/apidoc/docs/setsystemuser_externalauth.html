setsystemuser_externalauth

Use this method to set whether to use external authentication servers for the system user authentication or not

Syntax



Parameters

username

Name of the system-user entry to modify.
This is mandatory parameter.

externalauth

Whether to use external authentication servers for the system user authentication or not
Default value = ENABLED.
Possible Values : ENABLED, DISABLED.

Return Value

Returns simpleResult
