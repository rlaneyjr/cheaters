setrnat_srcippersistency

Use this method to set enables the NetScaler appliance to use the same NAT IP address for all RNAT sessions initiated from a particular server.

Syntax



Parameters

srcippersistency

Enables the NetScaler appliance to use the same NAT IP address for all RNAT sessions initiated from a particular server.
Default value = DISABLED.
Possible Values : ENABLED, DISABLED.

Return Value

Returns simpleResult
