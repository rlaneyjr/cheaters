unsetlbgroup_backuppersistencetimeout

Remove lb group backuppersistencetimeout setting.

Syntax



Parameters

name

Name of the lbgroup.
Default value = 0.

backuppersistencetimeout

Indicates that backuppersistencetimeout will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
