unsetauthenticationldapaction_ldaplogin

Remove authentication ldapAction ldaplogin setting.

Syntax



Parameters

name

Name of the authenticationldapAction.
Default value = 0.

ldaploginname

Indicates that ldaploginname will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
