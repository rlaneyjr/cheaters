setauthenticationauthnprofile_authenticationdomain

Use this method to set domain for which TM cookie must to be set. If unspecified, cookie will be set for FQDN.

Syntax



Parameters

name

Name of the authentication profile.
This is mandatory parameter.

authenticationdomain

Domain for which TM cookie must to be set. If unspecified, cookie will be set for FQDN.

Return Value

Returns simpleResult
