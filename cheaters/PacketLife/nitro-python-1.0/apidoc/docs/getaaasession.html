getaaasession

Use this method to get all AAA-TM/VPN connections that are bound to the specified user, group, IP address, or IP range.

Syntax



Parameters

username

Name of the AAA user.

groupname

Name of the AAA group.

intranetip

IP address or the first address in the intranet IP range.

netmask

Subnet mask for the intranet IP range.

Return Value

Returns getaaasessionResult
