unsetsnmptrap_community

Remove snmp trap community setting.

Syntax



Parameters

Name of the snmptrap.
Possible Values : generic, specific.

trapdestination

Name of the snmptrap.
Minimum value = 1.

communityname

Indicates that communityname will be reverted or reset to the default value.
