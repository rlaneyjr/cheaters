unsetgslbservice_sitepersistence

Remove gslb service sitepersistence setting.

Syntax



Parameters

servicename

Name of the gslbservice.
Default value = 0.

sitepersistence

Indicates that sitepersistence will be reverted or reset to the default value.
Default value = 0.
