unsetcsvserver_stateupdate

Remove cs vserver stateupdate setting.

Syntax



Parameters

name

Name of the csvserver.

stateupdate

Indicates that stateupdate will be reverted or reset to the default value.
Default value = 0.

Return Value

Returns simpleResult
