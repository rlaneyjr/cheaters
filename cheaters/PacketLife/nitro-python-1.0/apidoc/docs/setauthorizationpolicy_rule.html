setauthorizationpolicy_rule

Use this method to set name of the NetScaler named rule, or a default syntax expression, that the policy uses to perform the authentication.

Syntax



Parameters

name

Name of the authorization policy to modify.
This is mandatory parameter.

rule

Name of the NetScaler named rule, or a default syntax expression, that the policy uses to perform the authentication.

Return Value

Returns simpleResult
