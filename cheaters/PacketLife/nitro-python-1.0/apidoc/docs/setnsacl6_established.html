setnsacl6_established

Use this method to set allow only incoming TCP packets that have the ACK or RST bit set if the action set for the ACL6 rule is ALLOW and these packets match the other conditions in the ACL6 rule.

Syntax



Parameters

acl6name

Name of the ACL6 rule whose parameters you want to modify.
This is mandatory parameter.

established

Allow only incoming TCP packets that have the ACK or RST bit set if the action set for the ACL6 rule is ALLOW and these packets match the other conditions in the ACL6 rule.

Return Value

Returns simpleResult
