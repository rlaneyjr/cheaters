setvpnformssoaction_submitmethod

Use this method to set hTTP method (GET or POST) used by the single sign-on form to send the logon credentials to the logon server.

Syntax



Parameters

name

Name for the form based single sign-on profile.
This is mandatory parameter.

submitmethod

HTTP method (GET or POST) used by the single sign-on form to send the logon credentials to the logon server.
Default value = NS_ACT_FSSO_SUBMIT_GET.
Possible Values : GET, POST.

Return Value

Returns simpleResult
