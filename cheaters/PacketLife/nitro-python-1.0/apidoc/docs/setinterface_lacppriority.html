setinterface_lacppriority

Use this method to set lACP port priority, expressed as an integer. The lower the number, the higher the priority. The NetScaler appliance limits the number of interfaces in an LA channel to sixteen.

Syntax



Parameters

id

ID of the Interface whose parameters you want to modify. For a NetScaler appliance, specify the interface in C/U notation (for example, 1/3). For a cluster configuration, specify the interface in N/C/U notation (for example, 2/1/3). where C can take one of the following values: * 0 - Indicates a management interface. * 1 - Indicates a 1 Gbps port. * 10 - Indicates a 10 Gbps port. U is a unique integer for representing an interface in a particular port group. N is the ID of the node to which an interface belongs in a cluster configuration. Use spaces to separate multiple entries.
This is mandatory parameter.

lacppriority

LACP port priority, expressed as an integer. The lower the number, the higher the priority. The NetScaler appliance limits the number of interfaces in an LA channel to sixteen.
Default value = 32768.
Minimum value = 1.
Maximum value = 65535.

Return Value

Returns simpleResult
