bindbridgegroup_ipaddress

A network address or addresses to be associated with the bridge group. You must add entries for these network addresses in the routing table before running this method.

Syntax



Parameters

id

The integer that uniquely identifies the bridge group.
This is mandatory parameter.
Minimum value = 1.
Maximum value = 1000.

ipaddress

A network address or addresses to be associated with the bridge group. You must add entries for these network addresses in the routing table before running this method.

netmask

A subnet mask associated with the network address.

Return Value

Returns simpleResult
