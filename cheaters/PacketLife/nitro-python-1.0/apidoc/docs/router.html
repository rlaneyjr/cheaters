setrouterbgp_autonomoussystem

Use this method to set the autonomous system for BGP.
This is mandatory parameter.

setrouterbgp_conredistribute

Use this method to set the state of the router in redistributing connected routes. Use this option to enable the redistribution of connected routes into the BGP domain.

setrouterbgp_kernelredistribute

Use this method to set the state of the router in redistribution of kernel routes.

setrouterbgp_learnroute

Use this method to set the state of the router in learning routes from BGP. Use this option to enable route learning and installation from BGP.

setrouterbgp_neighbor

Use this method to set the IP address of a BGP peer for the router.

setrouterbgp_network

Use this method to set the network to be advertized.

setrouterbgp_routerid

Use this method to set the Router ID of this router.

setrouterbgp_staticredistribute

Use this method to set the state of the router in redistributing static routes. Use this option to enable the redistribution of static routes.

setrouterospf_conredistribute

Use this method to set the state of the router in redistributing connected routes. Use this option to enable the redistribution of connected routes.

setrouterospf_host

Use this method to set the stub link.

setrouterospf_kernelredistribute

Use this method to set the state of the router in redistributing kernel routes. Use this option to enable the redistribution of kernel routes.

setrouterospf_learnroute

Use this method to set the state of the router in learning routes from OSPF. Use this option to enable route learning from OSPF.

setrouterospf_network

Use this method to set the broadcast network on which OSPF is to be run.

setrouterospf_passiveinterface

Use this method to set the mode of the Interface. Use this option to change the mode of the interface to listen only.

setrouterospf_routerid

Use this method to set the router ID.

setrouterospf_staticredistribute

Use this method to set the state of the router in redistributing static routes. Use this option to enable the redistribution of static routes.

setrouterrip_defaultmetric

Use this method to set the default metrics when advertising routes.

setrouterrip_kernelredistribute

Use this method to set the state of redistributing kernel routes.

setrouterrip_learnroute

Use this method to set the state of Route learning. Use this option to enable route learning and installation in the kernel.

setrouterrip_network

Use this method to set the broadcast network on which RIP must run.

setrouterrip_passiveinterface

Use this method to set the mode of the interface to listen only.

setrouterrip_staticredistribute

Use this method to set the state of redistributing static routes.
