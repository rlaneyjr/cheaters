getdnsaaaarecEx

Use this method to get the AAAA (IPv6) address record for the specified host name. If a hostname is not specified, all configured AAAA records are shown.

Syntax



Parameters

hostname

Domain name.

ipv6address

One or more IPv6 addresses to assign to the domain name.

type

Type of records to display. Available settings function as follows: * ADNS - Display all authoritative address records. * PROXY - Display all proxy address records. * ALL - Display all address records.

Return Value

Returns getdnsaaaarecResult
