unsetipsecparameter_hashalgo

Remove ipsec parameter hashalgo setting.

Syntax



Parameters

hashalgo

Indicates that hashalgo will be reverted or reset to the default value.
Possible Values : HMAC_SHA1, HMAC_SHA256, HMAC_SHA384, HMAC_SHA512, HMAC_MD5.

Return Value

Returns simpleResult
