snmpcommunity module¶

class snmpcommunity¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for community resource.

communityname¶

The SNMP community string. Can consist of 1 to 31 characters that include uppercase and lowercase letters,numbers and special characters. The following requirement applies only to the NetScaler CLI: If the string includes one or more spaces, enclose the name in double or single quotation marks (for example, “my string” or ‘my string’).<br/>Minimum length = 1.

permissions¶

The SNMP V1 or V2 query-type privilege that you want to associate with this SNMP community.<br/>Possible values = GET, GET_NEXT, GET_BULK, SET, ALL.
