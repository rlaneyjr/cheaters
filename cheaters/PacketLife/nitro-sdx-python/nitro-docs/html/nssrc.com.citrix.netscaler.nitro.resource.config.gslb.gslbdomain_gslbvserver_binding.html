class gslbdomain_gslbvserver_binding¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Binding class showing the gslbvserver that can be bound to gslbdomain.

name¶

Name of the Domain.<br/>Minimum length = 1.

vservername¶
sitename¶

Name of the site to which the service belongs.

backuplbmethod¶

Indicates the backup method in case the primary fails.<br/>Possible values = ROUNDROBIN, LEASTCONNECTION, LEASTRESPONSETIME, SOURCEIPHASH, LEASTBANDWIDTH, LEASTPACKETS, STATICPROXIMITY, RTT, CUSTOMLOAD.

customheaders¶

The string that is sent to the service. Applicable to HTTP ,HTTP-ECV and RTSP monitor types.

persistenceid¶

Persistence id of the gslb vserver.

state¶

The state of the vserver.<br/>Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED.

lbmethod¶

The load balancing method set for the virtual server.<br/>Possible values = ROUNDROBIN, LEASTCONNECTION, LEASTRESPONSETIME, SOURCEIPHASH, LEASTBANDWIDTH, LEASTPACKETS, STATICPROXIMITY, RTT, CUSTOMLOAD.

edr¶

Send clients an empty DNS response when the GSLB virtual server is DOWN.<br/>Default value: DISABLED<br/>Possible values = ENABLED, DISABLED.

cip¶

Indicates if Client IP option is enabled.<br/>Possible values = ENABLED, DISABLED.

statechangetimesec¶

Time since last state change.

netmask¶

Netmask.<br/>Minimum length = 1.

dnsrecordtype¶

The IP type for this GSLB vserver.<br/>Possible values = A, AAAA, CNAME.

persistmask¶

The optional IPv4 network mask applied to IPv4 addresses to establish source IP address based persistence.<br/>Minimum length = 1.

sitepersistence¶

Indicates the type of cookie persistence set.<br/>Possible values = ConnectionProxy, HTTPRedirect, NONE.

servicetype¶

The type GSLB service.<br/>Possible values = HTTP, FTP, TCP, UDP, SSL, SSL_BRIDGE, SSL_TCP, NNTP, ANY, SIP_UDP, RADIUS, RDP, RTSP, MYSQL, MSSQL, ORACLE.

v6persistmasklen¶

Number of bits to consider in an IPv6 source IP address when creating source IP address based persistence sessions.<br/>Default value: 128<br/>Minimum value = 1<br/>Maximum value = 128.

v6netmasklen¶

Number of bits to consider, in an IPv6 source IP address, for creating the hash that is required by the SOURCEIPHASH load balancing method.<br/>Default value: 128<br/>Minimum value = 1<br/>Maximum value = 128.

persistencetype¶

Indicates if persistence is set on the gslb vserver.<br/>Possible values = SOURCEIP, NONE.

siteprefix¶

The site prefix string.

dynamicweight¶

Dynamic weight method of the vserver.<br/>Default value: DISABLED<br/>Possible values = SERVICECOUNT, SERVICEWEIGHT, DISABLED.

mir¶

.<br/>Default value: DISABLED<br/>Possible values = ENABLED, DISABLED.
