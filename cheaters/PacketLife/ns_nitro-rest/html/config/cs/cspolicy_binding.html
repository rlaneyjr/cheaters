cspolicy_binding

Binding object showing the resources that can be bound to cspolicy.

Read/write properties

policyname<String>

Name of the content switching policy to display. If this parameter is omitted, details of all the policies are displayed.
Minimum length = 1

Read only properties

cspolicy_cspolicylabel_binding<cspolicy_cspolicylabel_binding[]>

cspolicylabel that can be bound to cspolicy.

cspolicy_csvserver_binding<cspolicy_csvserver_binding[]>

csvserver that can be bound to cspolicy.

cspolicy_crvserver_binding<cspolicy_crvserver_binding[]>

crvserver that can be bound to cspolicy.
