crvserver_binding

Binding object showing the resources that can be bound to crvserver.

Read/write properties

name<String>

Name of a cache redirection virtual server about which to display detailed information.
Minimum length = 1

Read only properties

crvserver_filterpolicy_binding<crvserver_filterpolicy_binding[]>

filterpolicy that can be bound to crvserver.

crvserver_cmppolicy_binding<crvserver_cmppolicy_binding[]>

cmppolicy that can be bound to crvserver.

crvserver_lbvserver_binding<crvserver_lbvserver_binding[]>

lbvserver that can be bound to crvserver.

crvserver_policymap_binding<crvserver_policymap_binding[]>

policymap that can be bound to crvserver.

crvserver_cspolicy_binding<crvserver_cspolicy_binding[]>

cspolicy that can be bound to crvserver.

crvserver_crpolicy_binding<crvserver_crpolicy_binding[]>

crpolicy that can be bound to crvserver.
