vpnvserver_intranetip_binding

Binding object showing the intranetip that can be bound to vpnvserver.

Read/write properties

name<String>

Name of the virtual server.
Minimum length = 1

intranetip<String>

The network ID for the range of intranet IP addresses or individual intranet IP addresses to be bound to the virtual server.

netmask<String>

The netmask of the intranet IP address or range.

Read only properties

map<String>

Whether or not mapped IP addresses are ON or OFF. Mapped IP addresses are source IP addresses for the virtual servers running on the NetScaler. Mapped IP addresses are used by the system to connect to the backend servers.
Possible values = ON, OFF

acttype<Double>

.

__count<Double>

count parameter
