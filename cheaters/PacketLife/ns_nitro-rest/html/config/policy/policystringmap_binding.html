policystringmap_binding

Binding object showing the resources that can be bound to policystringmap.

Read/write properties

name<String>

Name of the string map to display. If a name is not provided, a list of all the configured string maps is shown.
Minimum length = 1

Read only properties

policystringmap_pattern_binding<policystringmap_pattern_binding[]>

pattern that can be bound to policystringmap.
