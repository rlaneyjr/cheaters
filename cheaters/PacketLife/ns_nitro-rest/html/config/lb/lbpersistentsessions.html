lbpersistentsessions

Configuration for persistence session resource.

Read/write properties

vserver<String>

The name of the virtual server.

persistenceparameter<String>

The persistence parameter whose persistence sessions are to be flushed.

Read only properties

type<Double>

Type of Persistence.

typestring<String>

Type of Persistence as String.

srcip<String>

SOURCE IP.

srcipv6<String>

SOURCE IPv6 ADDRESS.

destip<String>

DESTINATION IP.

destipv6<String>

DESTINATION IPv6 ADDRESS.

flags<Boolean>

IPv6 FLAGS.

destport<Integer>

Destination port.
Range 1 - 65535

vservername<String>

Virtual server name.

timeout<Double>

Persistent Session timeout.

referencecount<Double>

Reference Count.

persistenceparam<String>

Specific persistence information . Callid in case of SIP_CALLID persistence entry , RTSP session id in case of RTSP_SESSIONID persistence entry.

cnamepersparam<String>

The cname that is selected incase of gslb service.

__count<Double>

count parameter

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

warning   <String_value>

Specify to get warnings in nitro response. If this field is set to YES, warning message will be sent in 'message' field and 'WARNING' value is set in severity field of the response in case there is a warning.
Other possible values for severity are ERROR, INFO and NONE.
Default value: NO
Possible values = YES, NO

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE, ROLLBACK
