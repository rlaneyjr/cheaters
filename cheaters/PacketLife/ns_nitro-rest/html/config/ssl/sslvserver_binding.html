sslvserver_binding

Binding object showing the resources that can be bound to sslvserver.

Read/write properties

vservername<String>

Name of the SSL virtual server for which to show detailed information.
Minimum length = 1

Read only properties

sslvserver_sslpolicy_binding<sslvserver_sslpolicy_binding[]>

sslpolicy that can be bound to sslvserver.

sslvserver_sslcipher_binding<sslvserver_sslcipher_binding[]>

sslcipher that can be bound to sslvserver.

sslvserver_ecccurve_binding<sslvserver_ecccurve_binding[]>

ecccurve that can be bound to sslvserver.

sslvserver_sslcertkey_binding<sslvserver_sslcertkey_binding[]>

sslcertkey that can be bound to sslvserver.

sslvserver_sslciphersuite_binding<sslvserver_sslciphersuite_binding[]>

sslciphersuite that can be bound to sslvserver.
