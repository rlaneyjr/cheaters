sslservice_binding

Binding object showing the resources that can be bound to sslservice.

Read/write properties

servicename<String>

Name of the SSL service for which to show detailed information.
Minimum length = 1

Read only properties

sslservice_sslcipher_binding<sslservice_sslcipher_binding[]>

sslcipher that can be bound to sslservice.

sslservice_ecccurve_binding<sslservice_ecccurve_binding[]>

ecccurve that can be bound to sslservice.

sslservice_sslpolicy_binding<sslservice_sslpolicy_binding[]>

sslpolicy that can be bound to sslservice.

sslservice_sslcertkey_binding<sslservice_sslcertkey_binding[]>

sslcertkey that can be bound to sslservice.

sslservice_sslciphersuite_binding<sslservice_sslciphersuite_binding[]>

sslciphersuite that can be bound to sslservice.
