snmptrap_binding

Binding object showing the resources that can be bound to snmptrap.

Read/write properties

trapclass<String>

Trap type specified in the trap listener entry.
Possible values = generic, specific

trapdestination<String>

IP address specified in the trap listener entry.
Minimum length = 1

version<String>

The SNMP version of the trap specified in the trap listener entry.
Possible values = V1, V2, V3

td<Double>

Integer value that uniquely identifies the traffic domain in which you want to configure the entity. If you do not specify an ID, the entity becomes part of the default traffic domain, which has an ID of 0.
Minimum value = 0
Maximum value = 4094

Read only properties

snmptrap_snmpuser_binding<snmptrap_snmpuser_binding[]>

snmpuser that can be bound to snmptrap.
