bridgegroup

Configuration for bridge group resource.

Read/write properties

id<Double>

An integer that uniquely identifies the bridge group.
Minimum value = 1
Maximum value = 1000

ipv6dynamicrouting<String>

Enable all IPv6 dynamic routing protocols on all VLANs bound to this bridgegroup. Note: For the ENABLED setting to work, you must configure IPv6 dynamic routing protocols from the VTYSH command line.
Default value: DISABLED
Possible values = ENABLED, DISABLED

Read only properties

flags<Boolean>

Temporary flag used for internal purpose.

portbitmap<Double>

Member interfaces of this bridge group.

tagbitmap<Double>

Tagged members of this bridge group.

ifaces<String>

Names of all member interfaces of this bridge group.

tagifaces<String>

Names of all tagged member interfaces of this bridge group.

rnat<Boolean>

Temporary flag used for internal purpose.

__count<Double>

count parameter

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

warning   <String_value>

Specify to get warnings in nitro response. If this field is set to YES, warning message will be sent in 'message' field and 'WARNING' value is set in severity field of the response in case there is a warning.
Other possible values for severity are ERROR, INFO and NONE.
Default value: NO
Possible values = YES, NO

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE, ROLLBACK
