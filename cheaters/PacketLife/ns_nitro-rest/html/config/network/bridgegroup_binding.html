bridgegroup_binding

Binding object showing the resources that can be bound to bridgegroup.

Read/write properties

id<Double>

The name of the bridge group.
Minimum value = 1
Maximum value = 1000

Read only properties

bridgegroup_nsip_binding<bridgegroup_nsip_binding[]>

nsip that can be bound to bridgegroup.

bridgegroup_vlan_binding<bridgegroup_vlan_binding[]>

vlan that can be bound to bridgegroup.

bridgegroup_nsip6_binding<bridgegroup_nsip6_binding[]>

nsip6 that can be bound to bridgegroup.
