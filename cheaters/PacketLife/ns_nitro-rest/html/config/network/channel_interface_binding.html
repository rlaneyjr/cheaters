channel_interface_binding

Binding object showing the interface that can be bound to channel.

Read/write properties

id<String>

ID of the LA channel or the cluster LA channel to which you want to bind interfaces. Specify an LA channel in LA/x notation, where x can range from 1 to 8 or a cluster LA channel in CLA/x notation, where x can range from 1 to 4.

ifnum<String[]>

Interfaces to be bound to the LA channel of a NetScaler appliance or to the LA channel of a cluster configuration. For an LA channel of a NetScaler appliance, specify an interface in C/U notation (for example, 1/3). For an LA channel of a cluster configuration, specify an interface in N/C/U notation (for example, 2/1/3). where C can take one of the following values: * 0 - Indicates a management interface. * 1 - Indicates a 1 Gbps port. * 10 - Indicates a 10 Gbps port. U is a unique integer for representing an interface in a particular port group. N is the ID of the node to which an interface belongs in a cluster configuration. Use spaces to separate multiple entries.

Read only properties

lamode<String>

The mode(AUTO/MANNUAL) for the LA channel.
Possible values = MANUAL, AUTO

slaveduplex<Double>

Duplex of the member interfaces.

slaveflowctl<Double>

Flowcontrol of the member interfaces.

slavetime<Double>

UP time of the member interfaces.

slavestate<Double>

State of the member interfaces.

slavespeed<Double>

Speed of the member interfaces.

slavemedia<Double>

Media type of the member interfaces.

__count<Double>

count parameter
