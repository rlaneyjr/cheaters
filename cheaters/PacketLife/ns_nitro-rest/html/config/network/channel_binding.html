channel_binding

Binding object showing the resources that can be bound to channel.

Read/write properties

id<String>

ID of an LA channel or LA channel in cluster configuration whose details you want the NetScaler appliance to display. Specify an LA channel in LA/x notation, where x can range from 1 to 8 or a cluster LA channel in CLA/x notation, where x can range from 1 to 4.
Minimum length = 1

Read only properties

channel_interface_binding<channel_interface_binding[]>

interface that can be bound to channel.
