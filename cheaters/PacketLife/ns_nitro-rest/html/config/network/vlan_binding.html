vlan_binding

Binding object showing the resources that can be bound to vlan.

Read/write properties

id<Double>

Integer that uniquely identifies the VLAN for which the details are to be displayed.
Minimum value = 1
Maximum value = 4094

Read only properties

vlan_channel_binding<vlan_channel_binding[]>

channel that can be bound to vlan.

vlan_nsip_binding<vlan_nsip_binding[]>

nsip that can be bound to vlan.

vlan_interface_binding<vlan_interface_binding[]>

interface that can be bound to vlan.

vlan_linkset_binding<vlan_linkset_binding[]>

linkset that can be bound to vlan.

vlan_nsip6_binding<vlan_nsip6_binding[]>

nsip6 that can be bound to vlan.
