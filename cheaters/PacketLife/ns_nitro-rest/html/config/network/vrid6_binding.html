vrid6_binding

Binding object showing the resources that can be bound to vrid6.

Read/write properties

id<Double>

Integer value that uniquely identifies a VMAC6 address.
Minimum value = 1
Maximum value = 255

Read only properties

vrid6_nsip_binding<vrid6_nsip_binding[]>

nsip that can be bound to vrid6.

vrid6_interface_binding<vrid6_interface_binding[]>

interface that can be bound to vrid6.

vrid6_nsip6_binding<vrid6_nsip6_binding[]>

nsip6 that can be bound to vrid6.
