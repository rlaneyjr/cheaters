onlinkipv6prefix

Configuration for on-link IPv6 global prefixes for Router Advertisment resource.

Read/write properties

ipv6prefix<String>

Onlink prefixes for RA messages.

onlinkprefix<String>

RA Prefix onlink flag.
Default value: YES
Possible values = YES, NO

autonomusprefix<String>

RA Prefix Autonomus flag.
Default value: YES
Possible values = YES, NO

depricateprefix<String>

Depricate the prefix.
Default value: NO
Possible values = YES, NO

decrementprefixlifetimes<String>

RA Prefix Autonomus flag.
Default value: NO
Possible values = YES, NO

prefixvalidelifetime<Double>

Valide life time of the prefix, in seconds.
Default value: 2592000

prefixpreferredlifetime<Double>

Preferred life time of the prefix, in seconds.
Default value: 604800

Read only properties

prefixcurrvalidelft<Double>

Prefix current valid life time.

prefixcurrpreferredlft<Double>

Prefix current prefered life time.

__count<Double>

count parameter

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

warning   <String_value>

Specify to get warnings in nitro response. If this field is set to YES, warning message will be sent in 'message' field and 'WARNING' value is set in severity field of the response in case there is a warning.
Other possible values for severity are ERROR, INFO and NONE.
Default value: NO
Possible values = YES, NO

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE, ROLLBACK
