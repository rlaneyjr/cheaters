netbridge_binding

Binding object showing the resources that can be bound to netbridge.

Read/write properties

name<String>

The name of the network bridge.

Read only properties

netbridge_iptunnel_binding<netbridge_iptunnel_binding[]>

iptunnel that can be bound to netbridge.

netbridge_nsip_binding<netbridge_nsip_binding[]>

nsip that can be bound to netbridge.

netbridge_vlan_binding<netbridge_vlan_binding[]>

vlan that can be bound to netbridge.

netbridge_nsip6_binding<netbridge_nsip6_binding[]>

nsip6 that can be bound to netbridge.
