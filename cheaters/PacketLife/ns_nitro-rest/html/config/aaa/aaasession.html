aaasession

Configuration for active connection resource.

Read/write properties

username<String>

Name of the AAA user.
Minimum length = 1

groupname<String>

Name of the AAA group.
Minimum length = 1

iip<String>

IP address or the first address in the intranet IP range.
Minimum length = 1

netmask<String>

Subnet mask for the intranet IP range.
Minimum length = 1

all<Boolean>

Terminate all active AAA-TM/VPN sessions.

Read only properties

publicip<String>

Client's public IP address.

publicport<Integer>

Client's public port.
Range 1 - 65535

ipaddress<String>

NetScaler's IP address.

port<Integer>

NetScaler's port.
Range 1 - 65535

privateip<String>

Client's private/mapped IP address.

privateport<Integer>

Client's private/mapped port.
Range 1 - 65535

destip<String>

Destination IP address.

destport<Integer>

Destination port.
Range 1 - 65535

intranetip<String>

Specifies the Intranet IP.

peid<Double>

Core id of the session owner.

__count<Double>

count parameter

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

warning   <String_value>

Specify to get warnings in nitro response. If this field is set to YES, warning message will be sent in 'message' field and 'WARNING' value is set in severity field of the response in case there is a warning.
Other possible values for severity are ERROR, INFO and NONE.
Default value: NO
Possible values = YES, NO

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE, ROLLBACK
