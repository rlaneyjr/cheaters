systemuser_binding

Binding object showing the resources that can be bound to systemuser.

Read/write properties

username<String>

Name of a system user about whom to display information.
Minimum length = 1

Read only properties

systemuser_systemgroup_binding<systemuser_systemgroup_binding[]>

systemgroup that can be bound to systemuser.

systemuser_systemcmdpolicy_binding<systemuser_systemcmdpolicy_binding[]>

systemcmdpolicy that can be bound to systemuser.
