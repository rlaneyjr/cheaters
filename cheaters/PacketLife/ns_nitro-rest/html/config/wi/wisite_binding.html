wisite_binding

Binding object showing the resources that can be bound to wisite.

Read/write properties

sitepath<String>

Path of a Web Interface site whose details you want the NetScaler appliance to display.
Minimum length = 1
Maximum length = 250

Read only properties

wisite_translationinternalip_binding<wisite_translationinternalip_binding[]>

translationinternalip that can be bound to wisite.

wisite_accessmethod_binding<wisite_accessmethod_binding[]>

accessmethod that can be bound to wisite.

wisite_farmname_binding<wisite_farmname_binding[]>

farmname that can be bound to wisite.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.
