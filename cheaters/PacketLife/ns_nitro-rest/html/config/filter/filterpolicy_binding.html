filterpolicy_binding

Binding object showing the resources that can be bound to filterpolicy.

Read/write properties

name<String>

Name of the filter policy to be displayed. If a name is not provided, information about all the filter policies is shown.
Minimum length = 1

Read only properties

filterpolicy_csvserver_binding<filterpolicy_csvserver_binding[]>

csvserver that can be bound to filterpolicy.

filterpolicy_lbvserver_binding<filterpolicy_lbvserver_binding[]>

lbvserver that can be bound to filterpolicy.

filterpolicy_crvserver_binding<filterpolicy_crvserver_binding[]>

crvserver that can be bound to filterpolicy.

filterpolicy_filterglobal_binding<filterpolicy_filterglobal_binding[]>

filterglobal that can be bound to filterpolicy.
