gslbdomain_gslbservice_binding

Binding object showing the gslbservice that can be bound to gslbdomain.

Read/write properties

name<String>

Name of the Domain.
Minimum length = 1

servicename<String>

The service name.

Read only properties

cnameentry<String>

The cname of the gslb service.

state<String>

The state of the vserver.
Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED

dynamicconfwt<Double>

dynamic weight.

weight<Double>

weight assigned.
Minimum value = 1
Maximum value = 100

servicetype<String>

The type GSLB service.
Possible values = HTTP, FTP, TCP, UDP, SSL, SSL_BRIDGE, SSL_TCP, NNTP, ANY, SIP_UDP, RADIUS, RDP, RTSP, MYSQL, MSSQL, ORACLE

cumulativeweight<Double>

cumlative weight.

gslbthreshold<Integer>

The threshold value of the service.

port<Integer>

Port Number.
Minimum value = 1
Range 1 - 65535

vservername<String>

.

svreffgslbstate<String>

GSLB server state.
Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED

ipaddress<String>

The Ip address of the service.
