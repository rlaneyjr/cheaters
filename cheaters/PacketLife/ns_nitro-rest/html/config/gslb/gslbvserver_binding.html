gslbvserver_binding

Binding object showing the resources that can be bound to gslbvserver.

Read/write properties

name<String>

Name of the GSLB virtual server.
Minimum length = 1

Read only properties

gslbvserver_spilloverpolicy_binding<gslbvserver_spilloverpolicy_binding[]>

spilloverpolicy that can be bound to gslbvserver.

gslbvserver_gslbservice_binding<gslbvserver_gslbservice_binding[]>

gslbservice that can be bound to gslbvserver.

gslbvserver_domain_binding<gslbvserver_domain_binding[]>

domain that can be bound to gslbvserver.
