gslbsite_binding

Binding object showing the resources that can be bound to gslbsite.

Read/write properties

sitename<String>

Name of the GSLB site. If you specify a site name, details of all the site's constituent services are also displayed.
Minimum length = 1

Read only properties

gslbsite_gslbservice_binding<gslbsite_gslbservice_binding[]>

gslbservice that can be bound to gslbsite.
