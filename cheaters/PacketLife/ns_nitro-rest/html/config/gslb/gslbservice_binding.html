gslbservice_binding

Binding object showing the resources that can be bound to gslbservice.

Read/write properties

servicename<String>

Name of the GSLB service.
Minimum length = 1

Read only properties

gslbservice_lbmonitor_binding<gslbservice_lbmonitor_binding[]>

lbmonitor that can be bound to gslbservice.

gslbservice_dnsview_binding<gslbservice_dnsview_binding[]>

dnsview that can be bound to gslbservice.
