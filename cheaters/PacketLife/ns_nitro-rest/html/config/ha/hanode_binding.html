hanode_binding

Binding object showing the resources that can be bound to hanode.

Read/write properties

id<Double>

ID of the node whose HA settings you want to display. (The ID of the local node is always 0.).
Minimum value = 0
Maximum value = 64

Read only properties

hanode_partialfailureinterfaces_binding<hanode_partialfailureinterfaces_binding[]>

partialfailureinterfaces that can be bound to hanode.

hanode_routemonitor6_binding<hanode_routemonitor6_binding[]>

routemonitor6 that can be bound to hanode.

hanode_ci_binding<hanode_ci_binding[]>

ci that can be bound to hanode.

hanode_routemonitor_binding<hanode_routemonitor_binding[]>

routemonitor that can be bound to hanode.

hanode_fis_binding<hanode_fis_binding[]>

fis that can be bound to hanode.
