dnsview_binding

Binding object showing the resources that can be bound to dnsview.

Read/write properties

viewname<String>

Name of the view to display.
Minimum length = 1

Read only properties

dnsview_gslbservice_binding<dnsview_gslbservice_binding[]>

gslbservice that can be bound to dnsview.

dnsview_dnspolicy_binding<dnsview_dnspolicy_binding[]>

dnspolicy that can be bound to dnsview.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.
