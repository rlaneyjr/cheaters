dnszone_binding

Binding object showing the resources that can be bound to dnszone.

Read/write properties

zonename<String>

Name of the zone. Mutually exclusive with the type parameter.
Minimum length = 1

Read only properties

dnszone_domain_binding<dnszone_domain_binding[]>

domain that can be bound to dnszone.

dnszone_dnskey_binding<dnszone_dnskey_binding[]>

dnskey that can be bound to dnszone.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.
