server_gslbservice_binding

Binding object showing the gslbservice that can be bound to server.

Read/write properties

name<String>

Name of the server for which to display parameters.
Minimum length = 1

servicename<String>

The services attatched to the server.

Read only properties

serviceipstr<String>

This field has been intorduced to show the dbs services ip.

svctype<String>

The type of bound service.
Possible values = HTTP, FTP, TCP, UDP, SSL, SSL_BRIDGE, SSL_TCP, DTLS, NNTP, RPCSVR, DNS, ADNS, SNMP, RTSP, DHCPRA, ANY, SIP_UDP, DNS_TCP, ADNS_TCP, MYSQL, MSSQL, ORACLE, RADIUS, RDP, DIAMETER, SSL_DIAMETER, TFTP

svrstate<String>

The state of the bound service.
Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED

port<Integer>

The port number to be used for the bound service.
Range 1 - 65535

serviceipaddress<String>

The IP address of the bound service.

__count<Double>
