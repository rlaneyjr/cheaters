vlan

Statistics for "VLAN" resource.

Read/write properties

id<Double>

An integer specifying the VLAN identification number (VID). Possible values: 1 through 4094.
Minimum value = 1
Maximum value = 4094

clearstats<String>

Clear the statsistics / counters.
Possible values = basic, full

Read only properties

vlantotrxpkts<Double>

Packets received on the VLAN.

vlanrxpktsrate<Double>

Rate (/s) counter for vlantotrxpkts

vlantotrxbytes<Double>

Bytes of data received on the VLAN.

vlanrxbytesrate<Double>

Rate (/s) counter for vlantotrxbytes

vlantottxpkts<Double>

Packets transmitted on the VLAN.

vlantxpktsrate<Double>

Rate (/s) counter for vlantottxpkts

vlantottxbytes<Double>

Bytes of data transmitted on the VLAN.

vlantxbytesrate<Double>

Rate (/s) counter for vlantottxbytes

vlantotdroppedpkts<Double>

Inbound packets dropped by the VLAN upon reception.

vlantotbroadcastpkts<Double>

Broadcast packets sent and received on the VLAN.
