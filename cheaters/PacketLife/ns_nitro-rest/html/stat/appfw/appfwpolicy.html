appfwpolicy

Statistics for application firewall policy resource.

Read/write properties

name<String>

Name of the application firewall policy.
Minimum length = 1

clearstats<String>

Clear the statsistics / counters.
Possible values = basic, full

Read only properties

pipolicyhits<Double>

Number of hits on the policy

pipolicyhitsrate<Double>

Rate (/s) counter for pipolicyhits

pipolicyundefhits<Double>

Number of undef hits on the policy

pipolicyundefhitsrate<Double>

Rate (/s) counter for pipolicyundefhits
