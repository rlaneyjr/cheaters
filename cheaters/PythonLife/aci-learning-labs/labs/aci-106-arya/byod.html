Setting up the environment

This tutorial will instruct you in installing the Ayra Development Tool and creating a Python script using the tool.

Prerequisites

Python

The Python website offers two major versions of Python: 3.x.x and 2.x.x, and you will need 2.x.x for this lab. If you already have Python installed, you can check the version by simply running it from a terminal or shell:

C:\Users\Cisco>python
Python 2.7.9 (default, Dec 10 2014, 12:28:03) ...

Otherwise, you can download and install it from here:

https://www.python.org/downloads
