Setting up the environment

This tutorial will introduce you to the ACI command line interface (CLI) and the Application Program Interface (API). For this, you'll need an SSH client if you don't already have one.

Prerequisites

SSH

You will be logging in to the CLI using the SSH client.

For Windows, one of the most popular clients is Putty. It can be downloaded from:

    http://www.putty.org/

Because there is no installer, you will be downloading the binary .exe file itself. Therefore, you should put it in some sensible place (like C:Files (x86).exe), and make a shortcut to it on the desktop.

For Mac, the Terminal application is the most popular SSH client, and is already included with OS X.
