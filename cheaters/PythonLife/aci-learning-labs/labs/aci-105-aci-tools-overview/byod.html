Setting up the environment

There are no excercises in this particular tutorial, but it does provide instruction on downloading and installing the acitoolkit, which will be used in a subsequent lab. This will itself require Python and the Git application which must be installed and available on the client. The download locations for each of these applications is given below.

Prerequisites

Git

You will be performing a 'git clone' of the code from the Git Repository. There are a number of ways of achieving this. For example, an IDE, such as Eclipse, can do this for you. Otherwise, if you don't already have the command line version, you can download and install it from here:

http://git-scm.com/downloads

Python

The Python website offers two major versions of Python: 3.x.x and 2.x.x, and you will need 2.x.x for all ACI labs. If you already have Python installed, you can check the version by simply running it from a terminal or shell:

C:\Users\Cisco>python
Python 2.7.9 (default, Dec 10 2014, 12:28:03) ...

Otherwise, you can download and install it from here:

https://www.python.org/downloads

Get the acitoolkit library and tutorial

Next, you can download the the acitoolkit into a directory of your choice. This is done using the git utility installed above:

  git clone https://github.com/datacenter/acitoolkit.git

Install

Next, cd into the created directory and install the acitoolkit::

  cd acitoolkit
  python setup.py install

Note that when installing on Mac or Linux, you will likely need to run this as root so preface the command with the sudo keyword as follows::

  sudo python setup.py install

Setting login credentials

In the acitoolkit/samples directory there is a credentials.py file that contains the login credentials for the APIC and is shown (post edits) in its entirety below. It contains the following variables: LOGIN, PASSWORD, and IPADDR. These variables should be modified to the settings of your particular APIC.

   LOGIN = 'admin'
   PASSWORD = 'cisco123'
   IPADDR = '10.10.20.57'
   URL = 'https://' + IPADDR

Note: There may be another credentials.py file in the parent acitoolkit directory that is ignored and can be deleted. You should be modifying acitoolkit/samples/credentials.py.
