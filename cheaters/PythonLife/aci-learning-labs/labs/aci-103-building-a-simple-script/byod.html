Setting up the environment

This tutorial will be introducing you to the use of API Inspector to capture API calls in the GUI. It will also guide you through the creation of a Postman script in ACI.

Prerequisites

POSTMAN

You will be entering API calls using REST commands via a REST editor. One of the most popular Rest editors is POSTMAN, and it can be downloaded from:

    https://www.getpostman.com/apps

It also runs as an extension in the Chrome browser, which can be added by going to chrome://extensions/ and then clicking on the "Get more extensions" link at the bottom.

API Inspector

The API Inspector is included in the APIC GUI, and therefore no installation steps are required.
