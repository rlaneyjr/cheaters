Glob Qualifiers in Zsh

A glob qualifier is a brief set of characters in parentheses at the end of a pattern that provides some restriction on the type of file to be matched. Along with the **/ pattern, glob qualifiers are probably the most used extension to filename generation in zsh. A pair of parentheses at the end of a file pattern can include a short description of the type of file you want to match. The “short description” can be as simple as a dot:

ls *(.)

This picks out all regular files—no directories, symbolic links, or other special file types. Note this works only when the closing parenthesis is the final character of the pattern. In this section we cover the most common forms. See the manual (man zshexpn) for more.
