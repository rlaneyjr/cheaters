Summary

In this chapter we discussed the use of shell variables for storing different types of data. A scalar variable can hold a single value such as a string or a number, or you can use an array variable or associative array to store multiple values. The shell includes mechanisms for performing transformations on, searching for, and analyzing data stored in variables. In the following chapters, we will show how to combine many of these mechanisms to solve complex real problems. The next chapter explains how to write programs using the shell. These programs frequently make use of variables to store data.

This chapter has also given you an insight into the way the shell interprets what is on the command line. In particular we discussed how the contents of a command line are broken into separate words before being passed to a command. This is one of the more idiosyncratic aspects of the shell, but it has wide implications. A good understanding of this aspect of the shell will help you to avoid common mistakes.
