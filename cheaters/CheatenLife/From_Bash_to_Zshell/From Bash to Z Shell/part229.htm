Special Patterns in zsh

zsh’s additional globbing operators are in two classes: those usually enabled, and those for which the option extended_glob must be enabled. The first set extends the operation of char- acters already special to the shell, and are enabled by default since they aren’t likely to cause problems with compatibility. The second set uses characters that aren’t special inside patterns unless the option is set. First, we’ll talk about the ones available without a special option.
