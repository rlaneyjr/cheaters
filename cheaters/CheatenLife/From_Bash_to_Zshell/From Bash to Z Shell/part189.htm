The Zero Device

Although it’s less useful for shell programming, there is a special file /dev/zero that provides an infinite stream of null bytes. That’s an ASCII NUL, or character 0, not the character representing a printable zero in ASCII. We’ll illustrate this by running the command od (standing for “octal dump”) on the file. This shows the bytes in the file as a set of octal (base eight) numbers; here, as we’ve said, all the bytes are zero. The * indicates that the line above it is repeated. You will need to press Ctrl-c to stop the command endlessly reading zeros:

% od /dev/zero

0000000 000000 000000 000000 000000 000000 000000 000000 000000

*

This is very specialized; unless you are a system administrator, you are not likely to use

/dev/zero.
