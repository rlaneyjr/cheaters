bash

• Free Software Foundation’s web page: http://www.gnu.org/software/bash/bash.html

In addition to a brief summary of bash, this page contains links to an online version of the manual and download locations.

• The author’s web page: http://cnswww.cns.cwru.edu/~chet/bash/bashtop.html

Chet Ramey, who coordinates the development of bash, maintains this web page. It contains a good deal more information than the Free Software Foundation page.

• Frequently Asked Questions (FAQ): ftp://ftp.cwru.edu/pub/bash/FAQ

• Mailing list: bug-bash@gnu.org

There is only this one list and, though primarily intended for reporting bugs, it some- times contains discussion of new features or behavior changes; it is also available as a Usenet newsgroup, gnu.bash.bug.

• bash debugger: http://bashdb.sourceforge.net/

This is the web site for bashdb, which is covered in the section “The bash Debugger” in Chapter 13.

• bash_completion: http://www.caliban.org/bash/index.shtml#completion

To get the most out of programmable completion in bash, install this. See the section “bash_completion” in Chapter 10 for more details.
