Chapter 7: Prompts

When the shell needs to ask you something (for instance, “Do you want to change the spelling of this word?”) ortell you something (such as “You can enter a command line now”), it prompts you. Like almost everything about the shells, the prompt can be customized to show information you want, as you want it. This chapter shows you how.
