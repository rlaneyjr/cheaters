File Sizes

The letter L can be followed by a size in bytes to pick files of a particular size. You can think of it as standing for “length” (in bytes). It’s usually more useful to precede the size with a + or -, to specify files with a size greater than (or less than) the size in bytes. You can also change the units to kilobytes or megabytes by putting a k or m immediately after the L (and before any plus or minus sign).

Here, for example, are all the files in the zsh source directory that are larger than 100 kilo- bytes, excluding .o files:

zsh% ls -l ^*.o(Lk+100)
