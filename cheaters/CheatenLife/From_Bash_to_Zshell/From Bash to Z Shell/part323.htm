Complex Variable Expansions

As we have already seen, zsh allows special flags to be specified in variable expansions to do certain things like control how word splitting works. There are quite a few more things you can do from within variable expansions, including quite a few tricks that will work from older shells. We’ll start with these and move on to the more advanced features for which you’ll need zsh.
