More Complicated Qualifiers: String Arguments

A few qualifiers take a string as an argument. The argument is used to pass extra information to the qualifier, such as the username of a file’s owner or a specification for a file’s permissions. Because the syntax for qualifiers is rather terse, special care needs to be taken to mark where the argument starts and ends.
