More Shell Features


Chapter 1 introduced some related shell features and Unix concepts. Chapter 2 showed how to combine programs using redirection and shell programming features. In this chapter, we’ll

round out the book’s introduction by digging into a grab bag of more advanced shell and Unix features, including

• Passing information from process to process, such as the current time zone or the name of your favorite text editor, with environment variables

• Controlling processes: getting their status, stopping and starting them with job control, and sending signals to them

• Using quoting to control how the shell interprets a command line

• Changing your current directory by giving just the end of its pathname using the CDPATH
