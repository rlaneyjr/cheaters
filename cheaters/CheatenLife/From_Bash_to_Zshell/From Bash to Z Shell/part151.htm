Finding Out If You’re the Root User

In the previous example, we used a \$, not just a $. This displays a $ for most users, but it displays a # for the user called root, the system administrator. This lets you use the same prompt for everyone, but still be warned when you are running as root—an important fact, given all the damage you can do there if you are not careful. When run as root, the example would become the following:

bash$ PS1='\u@\h \$ '

root@pwstephenson #
