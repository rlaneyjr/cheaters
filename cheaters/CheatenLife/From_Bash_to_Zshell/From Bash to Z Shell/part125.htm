Summary

In this chapter we have learned the following:

• What bash and zsh do when they start

• How to control the shells by writing startup files

• How to make the shell execute commands in other files with the command “.”

• What login and interactive shells are

• How to control the shells with options; how to turn options on and off

In the next chapter we will further examine the command history. We showed in the last chapter how you can navigate through previous commands using the Up and Down Arrow keys. We’ll explain how the history is saved and restored, show how you can see other information about the commands you’ve typed, and demonstrate other shortcuts for accessing it.
