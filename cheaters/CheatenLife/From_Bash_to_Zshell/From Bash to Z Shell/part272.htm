Types of Completion

Completion isn’t limited to just adding extra missing characters. As we’ll see here, it can be configured to behave in a number of ways. In this section, we’ll introduce the following approaches:

• Menu completion allows you to cycle through the possible completion matches until you get to the one you want.

• Menu selection allows you to select a match from an interactive menu.

• Expansion takes such things as variable references and file-matching patterns and expands them in place on the command line. This allows you to see the arguments exactly as they will be passed on to the command when it is run.

• Approximate completion corrects any typos or mistakes in what you have already typed.
