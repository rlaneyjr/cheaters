More Unix Programs

This book is about shells, not about Unix in general. So we can’t go into detail about the literally hundreds of standard Unix programs available on a command line; for that, you’ll need a good introduction to Unix. Still, to use the power of a shell, you need to run programs! With that in mind, see Appendix A for a brief look at some common Unix programs, also called Unix utilities.
