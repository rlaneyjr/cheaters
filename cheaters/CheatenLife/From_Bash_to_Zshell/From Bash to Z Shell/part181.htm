Directories

We list directories first because the first thing you need to know about a file is what directory it’s in. By now you are familiar with the current directory, and how to change it with cd. We’ll reveal some more magic with cd below. You also know the difference between absolute file- names (those that begin with a / or ~) and relative filenames (those that don’t).
