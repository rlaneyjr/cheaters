Links

The link is a type of file many expert shell users find a big help. A link is a pointer to another file. More precisely, it is an entry in the filesystem that indicates where another file is to be found. Actually, there are two types of link on Unix systems, referred to as hard and symbolic (or soft) links. It’s quite important to know the differences between them.
